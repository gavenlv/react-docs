CSS 性能优化示例

1. 关键 CSS（Critical CSS）

将首屏渲染所需的 CSS 内联到 HTML 中
优化前： <head>
  <link rel="stylesheet" href="styles.css">
</head>

优化后： <head>
  <style>
    /* 关键 CSS 内联 */
    body { margin: 0; font-family: sans-serif; }
    .header { background: #2c3e50; color: white; }
  </style>
  <link rel="stylesheet" href="styles.css" media="print" onload="this.media='all'">
</head>
💡 提示：关键 CSS 可以减少首屏渲染时间，提升用户体验。

2. GPU 加速

使用 transform 和 opacity 触发 GPU 加速
避免使用（会触发重排）： .box {
  left: 100px; /* 触发布局重排 */
  top: 100px;
}

推荐使用（GPU 加速）： .box {
  transform: translate(100px, 100px); /* GPU 加速 */
}

强制 GPU 加速： .box {
  transform: translateZ(0);
  /* 或 */
  will-change: transform;
}
GPU 加速
will-change

3. will-change 属性

提前告知浏览器元素将发生变化
正确使用： .animated-element {
  will-change: transform, opacity;
}

⚠️ 不要过度使用： * {
  will-change: all; /* 不推荐 */
}
💡 提示：will-change 会消耗内存，只在需要时使用，动画结束后移除。

4. contain 属性

隔离页面的一部分，限制浏览器重排重绘范围
contain 值： .isolated-component {
  contain: layout; /* 布局隔离 */
}

.widget {
  contain: layout style; /* 布局和样式隔离 */
}

.card {
  contain: layout style paint; /* 完全隔离 */
}

这个容器使用了 contain: layout style

内部变化不会影响外部布局

5. content-visibility

跳过屏幕外内容的渲染工作
使用示例： .off-screen-content {
  content-visibility: auto;
  contain-intrinsic-size: 500px; /* 预估高度 */
}

这个容器使用了 content-visibility: auto

当不在视口内时，浏览器会跳过渲染

💡 提示：content-visibility 可以显著提升长页面的渲染性能。

6. 选择器性能

使用高效的选择器
✅ 好的选择器：
.nav-item - 简单类选择器
#header - ID 选择器
button.primary - 组合选择器
❌ 避免的选择器：
* { } - 通配符选择器
div div div p { } - 过深的嵌套
[type="text"] - 属性选择器（性能较低）

7. 动画性能优化

只动画 transform 和 opacity 属性
✅ 高性能动画属性： /* 这些属性不会触发重排 */
transform
opacity

❌ 避免动画这些属性： /* 会触发重排或重绘 */
width, height
top, left
margin, padding
background-color

8. 字体优化

使用 font-display 优化字体加载
@font-face 优化： @font-face {
  font-family: 'MyFont';
  src: url('font.woff2') format('woff2');
  font-display: swap; /* 立即显示后备字体 */
}

font-display 选项： /* swap: 立即显示后备字体 */
/* optional: 根据网速决定 */
/* block: 等待字体加载 */
💡 提示：使用 font-display: swap 可以避免 FOIT（Flash of Invisible Text）。

9. CSS 变量优化

使用 CSS 变量减少代码重复
定义变量： :root {
  --primary: #3498db;
  --spacing: 20px;
}

使用变量： .button {
  background: var(--primary);
  padding: var(--spacing);
}

使用 CSS 变量的优势：

  • 减少代码重复
  • 易于维护和主题切换
  • 浏览器原生支持，性能好

10. 布局优化

使用现代布局技术减少重排
✅ 推荐：Flexbox 和 Grid .container {
  display: flex;
  gap: 20px;
}

✅ 推荐：CSS Grid .grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
}
