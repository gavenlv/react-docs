CSS 盒模型示例

1. 标准盒模型 vs IE 盒模型

content-box（标准盒模型）
宽度 = 300px + 40px + 20px = 360px
实际宽度：360px
width(300) + padding(40) + border(20)
border-box（IE 盒模型）
宽度 = 300px（包含 padding 和 border）
实际宽度：300px
width 已包含 padding 和 border

2. 盒模型可视化

内容区域 (Content)
颜色说明：
绿色 = padding（内边距）
蓝色 = border（边框）
红色 = content（内容）
外部空白 = margin（外边距）
