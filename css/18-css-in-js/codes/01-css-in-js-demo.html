CSS-in-JS 示例

1. Styled Components

使用标签模板字符串创建样式化组件
React + Styled Components： import styled from 'styled-components';

// 创建样式化按钮组件
const Button = styled.button`
  background: #61dafb;
  color: #282c34;
  border: none;
  padding: 12px 30px;
  border-radius: 5px;
  cursor: pointer;
  transition: all 0.3s;

  &:hover {
    background: #21a1f1;
    transform: translateY(-2px);
  }
`;

// 使用组件
function App() {
  return <Button>Click Me</Button>;
}

2. CSS Modules

CSS 文件局部作用域，自动生成唯一类名
Card.module.css： .card {
  background: white;
  border-radius: 8px;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
}

.header {
  background: #61dafb;
  padding: 20px;
}

.body {
  padding: 20px;
}

Card.jsx： import styles from './Card.module.css';

function Card() {
  return (
    <div className={styles.card}>
      <div className={styles.header}>Header</div>
      <div className={styles.body}>Body</div>
    </div>
  );
}

CSS Modules 卡片

编译后的类名：Card_card__1a2b3

3. Emotion

灵活的 CSS-in-JS 库
React + Emotion： import { css } from '@emotion/react';

const style = css`
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;
  padding: 30px;
  border-radius: 8px;
`;

function GradientBox() {
  return <div css={style}>Gradient Box</div>;
}

5. 方案对比

不同 CSS-in-JS 方案的特点
方案 优点 缺点 适用场景
Styled Components 组件化、动态样式、主题支持 运行时开销、学习曲线 React 项目、动态主题
CSS Modules 局部作用域、零运行时、简单 不够灵活、静态样式 传统项目、静态样式
Emotion 灵活、性能好、SSR 支持 配置复杂 复杂应用、SSR 项目
Tailwind CSS 快速开发、一致性、体积小 类名冗长、定制成本 快速原型、团队协作

6. 动态样式示例

根据 props 动态改变样式
Styled Components 动态样式： const Button = styled.button`
  background: ${props => props.primary ? '#61dafb' : 'white'};
  color: ${props => props.primary ? 'white' : '#61dafb'};
  padding: 12px 30px;
`;

// 使用
<Button primary>Primary</Button>
<Button>Default</Button>
