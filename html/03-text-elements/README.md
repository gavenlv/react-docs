## 📰 标题标签

HTML 提供六级标题，`<h1>` 最大，`<h6>` 最小。

```html
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>
```

### 使用原则

- 每个页面通常只有一个 `<h1>`
- 按层级顺序使用，不要跳级
- 用于内容结构，不要仅用于样式

## 📝 段落和换行

```html
<p>这是一个段落。段落之间会有间距。</p>
<p>这是另一个段落。</p>

<p>第一行<br>第二行</p>

<hr>
<p>水平线用于分隔内容。</p>
```

---

## ✨ 文本格式化

### 语义化标签

```html
<strong>重要文本</strong>     <!-- 加粗，表示重要性 -->
<em>强调文本</em>             <!-- 斜体，表示强调 -->
<mark>高亮文本</mark>         <!-- 高亮显示 -->
<del>删除文本</del>           <!-- 删除线 -->
<ins>插入文本</ins>           <!-- 下划线 -->
<sub>下标</sub>               <!-- 下标 -->
<sup>上标</sup>               <!-- 上标 -->
<small>小号文本</small>       <!-- 小号文字 -->
```

### 非语义化标签

```html
<b>粗体文本</b>               <!-- 仅样式 -->
<i>斜体文本</i>               <!-- 仅样式 -->
<u>下划线文本</u>             <!-- 仅样式 -->
```

## 📋 引用和代码

```html
<!-- 块引用 -->
<blockquote>
  这是一段引用的文字。
  <cite>——作者名</cite>
</blockquote>

<!-- 短引用 -->
<p>他说：<q>学习 HTML 很有趣。</q></p>

<!-- 代码 -->
<code>console.log('Hello');</code>

<!-- 预格式化文本 -->
<pre>
  预格式化文本
  保留    空格和换行
</pre>

<!-- 键盘输入 -->
<kbd>Ctrl</kbd> + <kbd>C</kbd>

<!-- 变量 -->
<var>x</var> = <var>y</var> + 1
```

## 🔤 特殊字符

| 字符 | 实体 | 描述 |
|------|------|------|
| `<` | `&lt;` | 小于号 |
| `>` | `&gt;` | 大于号 |
| `&` | `&amp;` | 和号 |
| `"` | `&quot;` | 双引号 |
| `'` | `&apos;` | 单引号 |
| ` ` | `&nbsp;` | 不间断空格 |
| `©` | `&copy;` | 版权符号 |
| `®` | `&reg;` | 注册商标 |
| `™` | `&trade;` | 商标符号 |

## 💻 完整示例

```html
<article>
  <h1>文章标题</h1>
  
  <p>这是文章的<strong>重要内容</strong>，包含<em>强调的文字</em>。</p>
  
  <h2>章节标题</h2>
  <p>段落内容...</p>
  
  <blockquote>
    这是一段引用。
  </blockquote>
  
  <pre><code>
    function hello() {
      console.log('Hello, World!');
    }
  </code></pre>
</article>
```
