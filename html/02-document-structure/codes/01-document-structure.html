HTML 文档结构可视化

<html lang="zh-CN">
<head>
<meta charset="UTF-8"> - 字符编码
<meta name="viewport"> - 视口设置
<meta name="description"> - 页面描述
<title> - 页面标题
<link rel="stylesheet"> - 样式表
<body>
<header> - 页眉区域
<nav> - 导航区域
<main> - 主要内容
<footer> - 页脚区域

完整代码示例

<!-- 文档类型声明 -->
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>页面标题</title>
</head>
<body>
  <!-- 页面内容 -->
</body>
</html>
