# 19 - 嵌入内容

## 🎯 本节目标
- 掌握 iframe 的使用
- 了解 embed 和 object
- 学会嵌入第三方内容

---

## 📺 iframe

```html
<iframe 
  src="https://example.com" 
  width="600" 
  height="400"
  sandbox="allow-scripts allow-same-origin">
</iframe>
```

---

## 🔌 embed 和 object

```html
<embed src="file.pdf" type="application/pdf">
<object data="file.pdf" type="application/pdf"></object>
```
