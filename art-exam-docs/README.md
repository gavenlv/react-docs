# 广州市初中艺术学业水平考试教程

## 📚 考试概述

### 考试性质

初中学业水平考试是义务教育阶段的终结性考试，主要衡量学生达到国家规定学习要求的程度。艺术（含音乐和美术）属于录取参考科目。

### 考试对象

适用对象为广州市初中应届毕业生和报考广州市高中阶段学校的返穗生、往届生。

### 考试时间

- **艺术实践考察**：九年级上学期结束前进行
- **上机考试**：九年级下学期结束前进行

---

## 📊 考试结构

### 分值分布

| 科目 | 总分 | 上机考试 | 实践考察 |
|------|------|----------|----------|
| **艺术** | 100分 | 70分 | 30分 |
| 音乐 | 50分 | 35分 | 15分 |
| 美术 | 50分 | 35分 | 15分 |

### 上机考试结构

#### 音乐上机考试（35分）

| 题型 | 题量 | 分值 | 考试时长 |
|------|------|------|----------|
| 单项选择题 | 25题 | 25分 | 30分钟 |
| 判断题 | 10题 | 10分 | - |

#### 美术上机考试（35分）

| 题型 | 题量 | 分值 | 考试时长 |
|------|------|------|----------|
| 单项选择题 | 25题 | 25分 | 30分钟 |
| 判断题 | 10题 | 10分 | - |

### 实践考察

#### 音乐实践考察（15分）

- **考试内容**：从初中《音乐》教科书10首必唱教材歌曲中自选1首进行演唱
- **评分维度**：音准、节奏、歌词表达、流畅度、表现力

#### 美术实践考察（15分）

- **考试内容**：从初中《美术》教科书10个必画教材内容中自选1个进行绘画表现
- **评分维度**：构图、造型、色彩、创意、完成度

## 📈 成绩呈现与使用

### 成绩等级

录取参考科目成绩评定由高到低分为 **A、B、C、D、E** 五个等级：

| 等级 | 比例 |
|------|------|
| A | 25% |
| B | 35% |
| C | 25% |
| D | 10% |
| E | 5% |

### 成绩使用

| 录取类型 | 成绩要求 |
|----------|----------|
| 示范性普通高中 | 录取参考科目成绩均须达到 **C级及以上** |
| 非示范性普通高中 | 录取参考科目成绩均须达到 **D级及以上** |

## ⚠️ 高频考点速查

### 音乐高频考点

| 考点 | 内容 |
|------|------|
| 乐理基础 | 音符时值、节拍类型、调号识别 |
| 民族乐器 | 分类及代表乐器、代表作品 |
| 西洋乐器 | 管弦乐队编制、乐器组分类 |
| 京剧知识 | 行当分类、表演形式、代表人物 |
| 民歌分类 | 山歌、小调、劳动号子 |
| 经典作品 | 作者、乐器、风格特点 |

### 美术高频考点

| 考点 | 内容 |
|------|------|
| 美术门类 | 六大门类及特点 |
| 色彩知识 | 三要素、三原色、间色、互补色 |
| 透视知识 | 平行透视、成角透视、散点透视 |
| 构图形式 | 三角形、S形、水平线等构图特点 |
| 中国画 | 分类、特点、代表作品及作者 |
| 西方绘画 | 文艺复兴、印象派代表作品及作者 |
| 雕塑建筑 | 中外代表作品 |

## 📝 复习建议

### 系统复习

1. 按章节顺序系统学习
2. 完成每章单元测试
3. 整理错题，查漏补缺

### 重点突破

1. 熟记高频考点
2. 理解核心概念
3. 多做练习题

### 实践准备

1. 音乐：选择擅长的必唱歌曲，反复练习
2. 美术：选择擅长的必画内容，完成作品

### 模拟测试

1. 完成三套模拟试卷
2. 控制答题时间
3. 检验复习效果
