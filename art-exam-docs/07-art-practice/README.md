## 📋 第一部分：考试要求

### 1.1 考试内容

以《义务教育艺术课程标准（2022年版）》的目标、内容和学业要求为依据，从初中《美术》教科书**10个必画教材内容**中自选1个进行绘画表现。

### 1.2 考试方式

- 学校随堂组织绘画现场操作考试
- 学校提供统一规格的绘画纸
- 对每名考生绘画过程录像
- 满分：**15分**

### 1.3 评分维度

| 维度 | 分值 | 说明 |
|------|------|------|
| **构图** | 3分 | 画面布局合理，主次分明 |
| **造型** | 3分 | 形象准确，比例恰当 |
| **色彩** | 3分 | 色彩和谐，搭配得当 |
| **创意** | 3分 | 构思新颖，有个性表达 |
| **完成度** | 3分 | 画面完整，细节丰富 |

### 1.4 评分标准

| 等级 | 分值范围 | 标准 |
|------|----------|------|
| 优秀 | 13-15分 | 各维度表现优秀，画面完整，富有创意 |
| 良好 | 10-12分 | 各维度表现良好，基本完整，有一定创意 |
| 合格 | 7-9分 | 基本完成绘画，有少量失误 |
| 待提高 | 6分以下 | 绘画困难，失误较多 |

## 🖌️ 第二部分：绘画基础技能

### 2.1 素描基础

#### 线条练习

| 线条类型 | 练习方法 | 应用 |
|----------|----------|------|
| 直线 | 横、竖、斜线练习 | 轮廓线、结构线 |
| 曲线 | 圆弧、波浪线练习 | 曲面表现 |
| 排线 | 平行排线、交叉排线 | 明暗表现 |

#### 几何形体

```
基本几何形体：

  立方体      球体        圆柱体      圆锥体
   □          ○           ▭           △
```

| 形体 | 特点 | 明暗规律 |
|------|------|----------|
| 立方体 | 六个面，棱角分明 | 三大面明显 |
| 球体 | 圆滑过渡 | 明暗渐变 |
| 圆柱体 | 顶面圆形，侧面曲面 | 明暗过渡 |
| 圆锥体 | 底面圆形，侧面斜面 | 明暗渐变 |

#### 明暗素描步骤

```
步骤1：起稿
- 确定构图位置
- 画出基本轮廓

步骤2：铺大关系
- 确定明暗交界线
- 铺出大调子

步骤3：深入刻画
- 细化明暗层次
- 表现质感

步骤4：调整完善
- 整体调整
- 完善细节
```

### 2.2 色彩基础

#### 调色方法

| 调色 | 方法 | 效果 |
|------|------|------|
| 提高明度 | 加白色 | 颜色变亮 |
| 降低明度 | 加黑色 | 颜色变暗 |
| 降低纯度 | 加灰色或互补色 | 颜色变灰 |
| 调和颜色 | 两种以上颜色混合 | 产生新颜色 |

#### 色彩搭配

| 搭配方式 | 说明 | 效果 |
|----------|------|------|
| 同类色搭配 | 同一色相不同明度 | 和谐统一 |
| 邻近色搭配 | 色环相邻色彩 | 柔和协调 |
| 对比色搭配 | 色环相对色彩 | 鲜明对比 |
| 冷暖对比 | 冷色与暖色搭配 | 强烈对比 |

### 2.3 构图方法

#### 构图原则

| 原则 | 说明 |
|------|------|
| 主体突出 | 主要形象在视觉中心 |
| 大小适中 | 形象大小与画面协调 |
| 位置恰当 | 形象位置符合视觉规律 |
| 疏密有致 | 形象分布有聚有散 |

#### 常见构图形式

| 构图形式 | 适用场景 | 特点 |
|----------|----------|------|
| 中心构图 | 静物、人物 | 主体突出 |
| 三角形构图 | 静物、人物群像 | 稳定感强 |
| S形构图 | 风景、河流 | 流动感 |
| 对角线构图 | 动态场景 | 动感强 |

### 3.2 静物画

#### 题材内容

- 水果组合
- 花瓶与花卉
- 书本与文具
- 厨房用具

#### 绘画要点

| 要点 | 说明 |
|------|------|
| 构图 | 三角形构图，主次分明 |
| 造型 | 形体准确，比例恰当 |
| 色彩 | 色彩丰富，冷暖对比 |
| 质感 | 表现不同材质特点 |

#### 步骤示范

```
静物画步骤：

1. 观察 ──→ 分析静物组合关系
2. 起稿 ──→ 确定构图，画出轮廓
3. 铺色 ──→ 铺大色块
4. 深入 ──→ 刻画细节，表现质感
5. 调整 ──→ 整体调整，完善画面
```

### 3.3 风景画

#### 题材内容

- 自然风景
- 城市街景
- 校园一角
- 公园景色

#### 绘画要点

| 要点 | 说明 |
|------|------|
| 透视 | 运用透视规律表现空间 |
| 构图 | 合理安排景物位置 |
| 色彩 | 表现光影变化 |
| 层次 | 远中近层次分明 |

#### 透视运用

```
风景画透视：

近 ──→ 大、清晰、色彩鲜艳
        ↓
中 ──→ 中等、较清晰
        ↓
远 ──→ 小、模糊、色彩偏灰蓝
```

### 3.4 人物画

#### 题材内容

- 人物头像
- 人物半身像
- 人物全身像
- 人物动态

#### 绘画要点

| 要点 | 说明 |
|------|------|
| 比例 | 掌握人体比例关系 |
| 结构 | 理解人体结构 |
| 动态 | 表现人物动态 |
| 神态 | 刻画人物表情 |

#### 人体比例

```
人体比例（以头长为单位）：

成人全身：约7-7.5个头长
儿童全身：约5-6个头长

头部：1个头长
躯干：约2-2.5个头长
上肢：约3个头长
下肢：约4个头长

面部比例（三庭五眼）：
- 三庭：发际线到眉心、眉心到鼻底、鼻底到下巴
- 五眼：面部宽度约五个眼睛宽
```

### 3.5 花鸟画

#### 题材内容

| 设色 | 色彩淡雅，层次分明 |

| 花卉 | 特点 | 技法 |
|------|------|------|
| 牡丹 | 花瓣层次丰富 | 勾勒填色 |

| 梅花 | 枝干苍劲 | 勾花点蕊 |

## ⚠️ 常见问题与解决

### 问题1：构图问题

| 问题 | 解决方法 |
|------|----------|
| 画面太空 | 增加内容，丰富画面 |
| 画面太满 | 减少内容，留出空间 |
| 主体不突出 | 调整位置，突出主体 |
| 位置偏移 | 重新确定中心位置 |

### 问题2：造型问题

| 问题 | 解决方法 |
|------|----------|
| 比例失调 | 反复比较，调整比例 |
| 形象不准 | 多观察，多比较 |
| 结构错误 | 理解结构，正确表现 |
| 动态僵硬 | 抓住动态线，表现动感 |

### 问题3：色彩问题

| 问题 | 解决方法 |
|------|----------|
| 色彩脏 | 避免过多混色 |
| 色彩灰 | 增加纯色对比 |
| 色彩乱 | 确定主色调 |
| 色彩平 | 增加明暗变化 |

### 问题4：时间问题

| 问题 | 解决方法 |
|------|----------|
| 时间不够 | 合理分配时间，先整体后局部 |
| 画得太慢 | 多练习，提高速度 |
| 画得太快 | 放慢节奏，深入刻画 |
