| 要素 | 内容 |

| **旋律** | 音乐的主题线条 |

### 四、简答题

**1. 《义勇军进行曲》：**
- 音乐特点：进行曲风格，雄壮激昂，节奏有力
- 历史意义：激励中华儿女抗日救亡，后成为国歌

**2. 《二泉映月》：**
- 情感：深沉悲凉，如泣如诉
- 作者：阿炳（华彦钧）

**3. 《黄河大合唱》：**
- 乐章：《黄河船夫曲》《黄河颂》《保卫黄河》等
- 主题：中华民族不屈不挠的斗争精神

**4. 《天鹅湖》音乐特点：**
- 优美典雅，旋律动人
- 音乐与舞蹈完美结合
- 芭蕾舞剧经典之作

**5. 中国民歌：**
- 《茉莉花》- 江苏
- 《小河淌水》- 云南
- 《青春舞曲》- 新疆
