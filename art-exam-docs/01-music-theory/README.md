## 🎵 第一部分：音符与休止符

### 1.1 音符

**音符**是记录音的高低和长短的符号。

#### 常用音符

| 音符名称 | 五线谱记法 | 简谱记法 | 时值（以四分音符为一拍） |
|----------|------------|----------|--------------------------|
| 全音符 | 𝅝 | 5 --- | 4拍 |
| 二分音符 | 𝅗𝅥 | 5 - | 2拍 |
| 四分音符 | 𝅘𝅥 | 5 | 1拍 |
| 八分音符 | 𝅘𝅥𝅮 | 5 下方加一横 | 1/2拍 |
| 十六分音符 | 𝅘𝅥𝅯 | 5 下方加两横 | 1/4拍 |

#### 音符时值关系

```
全音符 = 2 × 二分音符
二分音符 = 2 × 四分音符
四分音符 = 2 × 八分音符
八分音符 = 2 × 十六分音符
```

#### 附点音符

**附点**是记在音符右边的小圆点，表示增加原音符时值的一半。

| 附点音符 | 时值计算 | 实际时值 |
|----------|----------|----------|
| 附点二分音符 | 2拍 + 1拍 | 3拍 |
| 附点四分音符 | 1拍 + 1/2拍 | 1.5拍 |
| 附点八分音符 | 1/2拍 + 1/4拍 | 3/4拍 |

### 1.2 休止符

**休止符**是表示音乐中停顿的符号。

| 休止符名称 | 五线谱记法 | 时值 |
|------------|------------|------|
| 全休止符 | 第四线下方一长横 | 4拍 |
| 二分休止符 | 第三线上方一长横 | 2拍 |
| 四分休止符 | 𝄽 | 1拍 |
| 八分休止符 | 𝄾 | 1/2拍 |
| 十六分休止符 | 𝄿 | 1/4拍 |

## 🥁 第二部分：节拍与节奏

### 2.1 节拍的概念

**节拍**是音乐中有规律的强弱交替。

#### 拍号

**拍号**用分数形式表示，写在乐曲开头。

```
拍号格式：分子/分母

分子：每小节的拍数
分母：以什么音符为一拍

例：2/4 表示以四分音符为一拍，每小节有两拍
```

### 2.2 常用拍子

| 拍号 | 含义 | 强弱规律 | 特点 |
|------|------|----------|------|
| 2/4 | 二拍子 | 强、弱 | 进行曲风格 |
| 3/4 | 三拍子 | 强、弱、弱 | 圆舞曲风格 |
| 4/4 | 四拍子 | 强、弱、次强、弱 | 常用拍子 |
| 3/8 | 三拍子 | 强、弱、弱 | 轻快活泼 |
| 6/8 | 六拍子 | 强、弱、弱、次强、弱、弱 | 复拍子 |

#### 拍子图示

**二拍子（2/4）**
```
强  弱
●   ○
```

**三拍子（3/4）**
```
强  弱  弱
●   ○   ○
```

**四拍子（4/4）**
```
强  弱  次强  弱
●   ○   ◐    ○
```

### 2.3 节奏与节拍的区别

| 概念 | 定义 | 特点 |
|------|------|------|
| **节拍** | 有规律的强弱交替 | 规律性、周期性 |
| **节奏** | 音的长短关系组合 | 多样性、变化性 |

**示例**：
- 节拍：强、弱、强、弱（固定规律）
- 节奏：X X XX X | X- X X | （长短变化）

### 2.4 切分音

**切分音**是改变正常强弱规律的节奏形式。

```
正常节奏：强 弱 | 强 弱
切分节奏：弱 强 | 弱 强

例：X X X → 中间的音变成强拍
```

## 🎼 第三部分：音乐记号

### 3.1 力度记号

**力度记号**表示音乐的强弱程度。

| 记号 | 意大利语 | 中文含义 |
|------|----------|----------|
| **pp** | pianissimo | 很弱 |
| **p** | piano | 弱 |
| **mp** | mezzo piano | 中弱 |
| **mf** | mezzo forte | 中强 |
| **f** | forte | 强 |
| **ff** | fortissimo | 很强 |

#### 力度变化记号

| 记号 | 含义 |
|------|------|
| **cresc.** 或 **\<** | 渐强 |
| **dim.** 或 **\>** | 渐弱 |

### 3.2 速度记号

**速度记号**表示音乐的快慢程度。

| 记号 | 意大利语 | 中文含义 | 每分钟拍数 |
|------|----------|----------|------------|
| **Grave** | - | 庄板 | 40-44 |
| **Largo** | - | 广板 | 46-50 |
| **Lento** | - | 慢板 | 52-54 |
| **Adagio** | - | 柔板 | 56-58 |
| **Andante** | - | 行板 | 66-69 |
| **Moderato** | - | 中板 | 88-104 |
| **Allegro** | - | 快板 | 112-116 |
| **Presto** | - | 急板 | 168-200 |

#### 速度变化记号

| 记号 | 含义 |
|------|------|
| **rit.** (ritardando) | 渐慢 |
| **accel.** (accelerando) | 渐快 |
| **a tempo** | 回原速 |

### 3.3 表情记号

| 记号 | 含义 |
|------|------|
| **dolce** | 柔美地 |
| **espressivo** | 有表情地 |
| **cantabile** | 如歌地 |
| **legato** | 连贯地 |
| **staccato** | 断奏、跳音 |
| **marcato** | 强调地 |

### 3.4 反复记号

| 记号 | 含义 |
|------|------|
| **\|\|:\|** | 从此处反复 |
| **\|:\|\|** | 反复到此结束 |
| **D.C.** (Da Capo) | 从头反复 |
| **D.S.** (Dal Segno) | 从记号处反复 |
| **Fine** | 结束 |
| **𝄋** | 反复跳跃记号 |

### 3.5 装饰音

| 名称 | 记号 | 说明 |
|------|------|------|
| 倚音 | 小音符 | 快速演奏，占主音时值 |
| 波音 | 𝆦 | 主音-上方音-主音 |
| 颤音 | tr | 主音与上方音快速交替 |
| 滑音 | ↘ 或 ↗ | 从一音滑向另一音 |

## 🎹 第四部分：调式与调号

### 4.1 音阶

**音阶**是按照音高顺序排列的一组音。

#### C大调音阶

```
音名：C  D  E  F  G  A  B  C
唱名：do re mi fa sol la si do
简谱：1  2  3  4  5  6  7  1
```

#### 大调音阶结构

```
全音 - 全音 - 半音 - 全音 - 全音 - 全音 - 半音
```

### 4.2 调号

**调号**是写在谱号后面，表示乐曲调性的升降记号。

#### 常用调号

| 调名 | 调号 | 升降音 |
|------|------|--------|
| C大调 | 无升降 | - |
| G大调 | 1个升号 | F# |
| D大调 | 2个升号 | F#、C# |
| F大调 | 1个降号 | B♭ |
| B♭大调 | 2个降号 | B♭、E♭ |

#### 升号调顺序

```
升号顺序：F → C → G → D → A → E → B
口诀：法 国 革 命 打 败 敌 人
```

#### 降号调顺序

```
降号顺序：B → E → A → D → G → C → F
口诀：必 须 按 时 到 广 场 吃 饭
```

### 4.3 中国民族调式

**五声调式**是中国传统音乐的基础。

#### 五声音阶

| 音名 | 唱名 | 特点 |
|------|------|------|
| 宫 | do | 主音，稳定 |
| 商 | re | - |
| 角 | mi | - |
| 徵 | sol | - |
| 羽 | la | - |

```
五声音阶特点：没有半音（fa和si）
音程关系：宫-商-角-徵-羽-宫
         全-全-小三-全-小三
```

#### 五声调式分类

| 调式 | 主音 | 结束音 |
|------|------|--------|
| 宫调式 | 宫(do) | 结束在do |
| 商调式 | 商(re) | 结束在re |
| 角调式 | 角(mi) | 结束在mi |
| 徵调式 | 徵(sol) | 结束在sol |
| 羽调式 | 羽(la) | 结束在la |

## 📊 第五部分：五线谱识读

### 5.1 五线谱基础

```
五线谱由五条平行的横线组成

    第五线 ─────────────
    第四线 ─────────────
    第三线 ─────────────
    第二线 ─────────────
    第一线 ─────────────
    
从下往上数：第一线、第二线、第三线、第四线、第五线
线与线之间称为"间"：第一间、第二间、第三间、第四间
```

### 5.2 谱号

#### 高音谱号（G谱号）

```
    𝄞
用于记录较高音域的乐曲
第二线为G音(sol)
```

#### 低音谱号（F谱号）

```
    𝄢
用于记录较低音域的乐曲
第四线为F音(fa)
```

### 5.3 音高位置

#### 高音谱表音高

```
上加一线：la (A)
第五线：fa (F)
第四间：mi (E)
第四线：re (D)
第三间：do (C)
第三线：si (B)
第二间：sol (G)
第二线：fa (F)
第一间：mi (E)
第一线：re (D)
下加一线：do (C) - 中央C
```

### 5.4 变音记号

| 记号 | 名称 | 作用 |
|------|------|------|
| ♯ | 升号 | 升高半音 |
| ♭ | 降号 | 降低半音 |
| ♮ | 还原号 | 还原到原音高 |

### 四、简答题

**1. 音符时值倍数关系：**
- 全音符 = 2×二分音符
- 二分音符 = 2×四分音符
- 四分音符 = 2×八分音符
- 八分音符 = 2×十六分音符

**2. 拍子区别：**
- 2/4拍：二拍子，强、弱，进行曲风格
- 3/4拍：三拍子，强、弱、弱，圆舞曲风格
- 4/4拍：四拍子，强、弱、次强、弱，常用拍子

**3. 力度记号含义：**
- pp = pianissimo = 很弱
- p = piano = 弱
- mf = mezzo forte = 中强
- f = forte = 强
- ff = fortissimo = 很强

**4. 五声调式特点：**
- 五声调式由宫、商、角、徵、羽五个音组成
- 特点是没有半音（缺少fa和si）
- 音程关系为全-全-小三-全-小三

**5. 变音记号作用：**
- 升号(♯)：将原音升高半音
- 降号(♭)：将原音降低半音
- 还原号(♮)：将升降的音还原到原音高
