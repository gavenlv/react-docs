## 📽️ 第一部分：PowerPoint 简介

### 1.1 PowerPoint 概述

**Microsoft PowerPoint** 是微软公司开发的演示文稿软件，用于制作电子幻灯片，广泛应用于教学、演讲、汇报等场合。

#### PowerPoint 的主要功能

| 功能 | 说明 |
|------|------|
| 制作幻灯片 | 创建演示文稿 |
| 添加内容 | 文字、图片、表格、图表等 |
| 设置动画 | 让内容动起来 |
| 设置切换 | 幻灯片之间的过渡效果 |
| 放映演示 | 全屏播放演示文稿 |

### 1.2 PowerPoint 界面

### 1.3 基本概念

| 概念 | 说明 |
|------|------|
| 演示文稿 | PowerPoint 文件，扩展名 .pptx |
| 幻灯片 | 演示文稿中的每一页 |
| 版式 | 幻灯片的布局格式 |
| 占位符 | 预留的内容区域 |
| 母版 | 统一设置幻灯片格式 |

### 1.4 视图模式

| 视图 | 说明 | 用途 |
|------|------|------|
| 普通视图 | 默认视图，编辑幻灯片 | 编辑内容 |
| 大纲视图 | 显示文字大纲 | 整理结构 |
| 幻灯片浏览 | 显示所有幻灯片缩略图 | 排序、删除 |
| 阅读视图 | 窗口放映 | 预览效果 |
| 幻灯片放映 | 全屏放映 | 正式演示 |

## 📄 第二部分：幻灯片操作

### 2.1 新建演示文稿

#### 方法一：新建空白演示文稿

1. 点击【文件】→【新建】
2. 选择"空白演示文稿"
3. 点击创建

#### 方法二：使用模板

1. 点击【文件】→【新建】
2. 选择模板或搜索模板
3. 点击创建

### 2.2 幻灯片基本操作

#### 新建幻灯片

| 方法 | 操作 |
|------|------|
| 功能区 | 开始 → 新建幻灯片 → 选择版式 |
| 快捷键 | Ctrl + M |
| 右键菜单 | 在缩略图区域右键 → 新建幻灯片 |

#### 删除幻灯片

| 方法 | 操作 |
|------|------|
| 快捷键 | 选中幻灯片 → Delete |
| 右键菜单 | 右键幻灯片 → 删除幻灯片 |

#### 复制幻灯片

| 方法 | 操作 |
|------|------|
| 快捷键 | Ctrl + D |
| 右键菜单 | 右键幻灯片 → 复制幻灯片 |

#### 移动幻灯片

- 在缩略图区域拖动幻灯片到目标位置

### 2.3 幻灯片版式

#### 常用版式

| 版式 | 说明 | 用途 |
|------|------|------|
| 标题幻灯片 | 只有大标题和副标题 | 封面 |
| 标题和内容 | 标题 + 内容区 | 图文混排 |
| 两栏内容 | 左右两栏 | 对比展示 |
| 比较 | 左右对比 | 对比分析 |
| 仅标题 | 只有标题 | 自由设计 |
| 空白 | 无占位符 | 完全自由设计 |

#### 更改版式

1. 选中幻灯片
2. 点击【开始】→【版式】
3. 选择新的版式

## 📝 第三部分：内容编辑

### 3.1 添加文本

#### 使用占位符

1. 点击占位符区域
2. 输入文字内容

#### 使用文本框

1. 点击【插入】→【文本框】
2. 选择横排或竖排文本框
3. 在幻灯片上拖动绘制
4. 输入文字

#### 文本格式设置

| 设置 | 方法 |
|------|------|
| 字体 | 开始 → 字体 |
| 字号 | 开始 → 字号 |
| 颜色 | 开始 → 字体颜色 |
| 加粗 | Ctrl + B |
| 倾斜 | Ctrl + I |
| 下划线 | Ctrl + U |

### 3.2 插入图片

#### 插入本地图片

1. 点击【插入】→【图片】→【此设备】
2. 选择图片文件
3. 点击插入

#### 插入在线图片

1. 点击【插入】→【图片】→【联机图片】
2. 搜索或选择图片
3. 点击插入

#### 图片编辑

| 操作 | 方法 |
|------|------|
| 调整大小 | 拖动角点或边中点 |
| 裁剪 | 格式 → 裁剪 |
| 旋转 | 拖动旋转手柄 |
| 样式 | 格式 → 图片样式 |

### 3.3 插入形状

#### 插入方法

1. 点击【插入】→【形状】
2. 选择形状类型
3. 在幻灯片上拖动绘制

#### 形状类型

| 类型 | 包含形状 |
|------|----------|
| 最近使用的形状 | 最近使用过的形状 |
| 线条 | 直线、箭头、曲线 |
| 基本形状 | 矩形、圆形、三角形 |
| 箭头 | 各种箭头 |
| 流程图 | 流程图符号 |
| 标注 | 气泡、标注框 |

#### 形状编辑

| 操作 | 方法 |
|------|------|
| 添加文字 | 右键形状 → 编辑文字 |
| 设置填充 | 格式 → 形状填充 |
| 设置轮廓 | 格式 → 形状轮廓 |
| 设置效果 | 格式 → 形状效果 |

### 3.4 插入表格

#### 插入方法

1. 点击【插入】→【表格】
2. 选择行列数或插入表格
3. 输入内容

#### 表格编辑

| 操作 | 方法 |
|------|------|
| 添加行列 | 布局 → 在左侧/右侧插入 |
| 删除行列 | 布局 → 删除 |
| 合并单元格 | 布局 → 合并单元格 |
| 设置样式 | 设计 → 表格样式 |

### 3.5 插入图表

#### 插入方法

1. 点击【插入】→【图表】
2. 选择图表类型
3. 编辑数据
4. 关闭数据窗口

#### 图表类型

| 图表 | 用途 |
|------|------|
| 柱形图 | 比较数据大小 |
| 折线图 | 显示变化趋势 |
| 饼图 | 显示占比关系 |
| 条形图 | 横向比较数据 |

### 3.6 插入音频和视频

#### 插入音频

1. 点击【插入】→【音频】→【PC 上的音频】
2. 选择音频文件
3. 设置播放选项

#### 插入视频

1. 点击【插入】→【视频】→【此设备】
2. 选择视频文件
3. 调整大小和位置

## 🎨 第四部分：设计美化

### 4.1 主题

#### 应用主题

1. 点击【设计】→【主题】
2. 选择主题样式
3. 点击应用

#### 自定义主题

| 设置 | 方法 |
|------|------|
| 颜色 | 设计 → 变体 → 颜色 |
| 字体 | 设计 → 变体 → 字体 |
| 效果 | 设计 → 变体 → 效果 |

### 4.2 背景

#### 设置背景

1. 点击【设计】→【设置背景格式】
2. 选择填充类型：
   - 纯色填充
   - 渐变填充
   - 图片或纹理填充
   - 图案填充

#### 背景样式

1. 点击【设计】→【背景样式】
2. 选择预设样式

### 4.3 幻灯片母版

#### 打开母版视图

1. 点击【视图】→【幻灯片母版】
2. 编辑母版
3. 点击【关闭母版视图】

#### 母版的作用

- 统一设置所有幻灯片的格式
- 添加统一的标题样式
- 添加统一的页脚、页码
- 添加统一的背景

## ✨ 第五部分：动画设置

### 5.1 动画类型

| 类型 | 说明 | 效果示例 |
|------|------|----------|
| 进入 | 对象出现 | 淡入、飞入、缩放 |
| 强调 | 对象突出 | 放大、旋转、变色 |
| 退出 | 对象消失 | 淡出、飞出、缩放 |
| 动作路径 | 对象移动 | 直线、曲线、自定义 |

### 5.2 添加动画

#### 基本步骤

1. 选中要添加动画的对象
2. 点击【动画】→【添加动画】
3. 选择动画效果

#### 常用动画效果

| 类型 | 效果 |
|------|------|
| 进入 | 淡入、飞入、浮入、劈裂、百叶窗 |
| 强调 | 放大/缩小、旋转、脉冲、变色 |
| 退出 | 淡出、飞出、浮出、劈裂 |

### 5.3 动画设置

#### 动画窗格

1. 点击【动画】→【动画窗格】
2. 查看和管理所有动画

#### 动画选项

| 设置 | 说明 |
|------|------|
| 开始 | 单击时、与上一动画同时、上一动画之后 |
| 持续时间 | 动画播放时长 |
| 延迟 | 延迟播放时间 |

#### 动画顺序

- 在动画窗格中拖动调整顺序
- 或使用"重新排序"按钮

### 5.4 幻灯片切换

#### 添加切换效果

1. 选中幻灯片
2. 点击【切换】→【切换效果】
3. 选择切换效果

#### 切换效果类型

| 类型 | 效果 |
|------|------|
| 细微型 | 淡出、推进、擦除 |
| 华丽型 | 翻转、库、百叶窗 |
| 动态内容 | 平移、形状、随机线条 |

#### 切换设置

| 设置 | 说明 |
|------|------|
| 效果选项 | 切换方向或样式 |
| 持续时间 | 切换动画时长 |
| 声音 | 切换时的声音 |
| 换片方式 | 单击鼠标或自动换片 |

#### 应用到所有幻灯片

- 点击【全部应用】按钮

## 🎬 第六部分：放映设置

### 6.1 开始放映

| 方法 | 操作 |
|------|------|
| 从头开始 | F5 或 幻灯片放映 → 从头开始 |
| 从当前页 | Shift + F5 或 幻灯片放映 → 从当前幻灯片开始 |

### 6.2 放映控制

#### 放映中操作

| 操作 | 快捷键 |
|------|--------|
| 下一页 | 空格、→、↓、Enter、PageDown |
| 上一页 | ←、↑、PageUp |
| 跳转到指定页 | 数字 + Enter |
| 结束放映 | Esc 或 - |
| 黑屏 | B 或 . |
| 白屏 | W 或 , |
| 暂停/继续 | S |
| 显示激光笔 | Ctrl + 鼠标左键 |

### 6.3 排练计时

#### 设置排练计时

1. 点击【幻灯片放映】→【排练计时】
2. 按照实际演讲速度放映
3. 记录每张幻灯片的时间
4. 保存排练时间

### 6.4 录制幻灯片

#### 录制演示

1. 点击【幻灯片放映】→【录制幻灯片演示】
2. 选择录制范围
3. 开始录制
4. 录制旁白和激光笔动作

### 6.5 放映设置

#### 设置放映方式

1. 点击【幻灯片放映】→【设置幻灯片放映】
2. 设置放映类型：
   - 演讲者放映（全屏幕）
   - 观众自行浏览（窗口）
   - 在展台浏览（全屏幕）

#### 放映选项

| 选项 | 说明 |
|------|------|
| 循环放映 | 放映结束后重新开始 |
| 放映时不加旁白 | 不播放录制的旁白 |
| 放映时不加动画 | 不播放动画效果 |

### 6.6 隐藏幻灯片

1. 选中幻灯片
2. 点击【幻灯片放映】→【隐藏幻灯片】
3. 放映时跳过该幻灯片

### 四、操作题

**操作题1 参考步骤：**

1. 新建演示文稿
2. 第 1 张幻灯片使用标题幻灯片版式，输入标题和副标题
3. 点击【开始】→【新建幻灯片】→ 选择标题和内容版式
4. 输入标题和内容
5. 新建第 3 张幻灯片，选择两栏内容版式
6. 输入标题，插入图片

**操作题2 参考步骤：**

1. 选中第 2 张幻灯片的标题
2. 点击【动画】→【添加动画】→ 选择"飞入"
3. 选中内容文本框
4. 点击【动画】→【添加动画】→ 选择"淡入"
5. 打开动画窗格，调整动画顺序

**操作题3 参考步骤：**

1. 点击【切换】→ 选择"淡出"效果
2. 点击【全部应用】
3. 在计时组中勾选"设置自动换片时间"，输入 00:05:00
4. 点击【幻灯片放映】→【设置幻灯片放映】
5. 勾选"循环放映，按 ESC 键终止"
