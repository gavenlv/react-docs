### 1.1 常见网络安全威胁

#### 威胁类型概述

| 威胁类型 | 说明 | 危害程度 |
|----------|------|----------|
| 计算机病毒 | 破坏计算机功能或数据的程序 | ⭐⭐⭐⭐⭐ |
| 木马程序 | 伪装成正常程序，窃取信息 | ⭐⭐⭐⭐⭐ |
| 网络钓鱼 | 伪装成可信网站骗取信息 | ⭐⭐⭐⭐ |

| 网络诈骗 | 通过网络骗取财物 | ⭐⭐⭐⭐ |

### 1.2 计算机病毒

#### 什么是计算机病毒

**计算机病毒**是一种能够自我复制、传播，对计算机系统造成破坏的程序代码。

#### 病毒的特点

| 特点 | 详细说明 | 示例 |
|------|----------|------|
| 传染性 | 能自我复制并传播到其他文件或系统 | 通过U盘、网络传播 |
| 破坏性 | 破坏数据、占用资源、导致系统崩溃 | 删除文件、格式化硬盘 |
| 隐蔽性 | 伪装成正常程序，不易被发现 | 隐藏在游戏、工具软件中 |
| 潜伏性 | 感染后可能不立即发作，等待触发条件 | 特定日期发作 |
| 可触发性 | 满足特定条件时激活 | 日期、时间、操作触发 |

#### 病毒传播途径

| 途径 | 说明 | 防范措施 |
|------|------|----------|
| 移动存储设备 | U盘、移动硬盘等 | 使用前先杀毒 |
| 网络下载 | 下载的文件携带病毒 | 从正规网站下载 |
| 电子邮件 | 邮件附件携带病毒 | 不打开陌生邮件附件 |
| 网页浏览 | 恶意网站植入病毒 | 不访问可疑网站 |
| 即时通讯 | 接收的文件携带病毒 | 不接收陌生人文件 |

#### 常见病毒类型

| 类型 | 特点 | 示例 |
|------|------|------|
| 文件型病毒 | 感染可执行文件 | CIH病毒 |
| 引导型病毒 | 感染系统引导区 | 小球病毒 |
| 宏病毒 | 感染Office文档 | Word宏病毒 |
| 蠕虫病毒 | 通过网络自我传播 | 熊猫烧香 |
| 脚本病毒 | 通过脚本语言传播 | VBS脚本病毒 |

### 1.3 木马程序

#### 什么是木马

**木马（Trojan）**是一种伪装成正常程序的恶意软件，不会自我复制，但会窃取用户信息或给黑客留下后门。

#### 木马与病毒的区别

| 特性 | 计算机病毒 | 木马程序 |
|------|------------|----------|
| 自我复制 | 能 | 不能 |
| 主要目的 | 破坏数据 | 窃取信息 |
| 传播方式 | 自动传播 | 需用户运行 |
| 隐蔽性 | 较强 | 非常强 |

#### 木马的危害

1. **窃取账号密码**：盗取游戏账号、网银密码等
2. **监控用户操作**：记录键盘输入、截屏
3. **远程控制**：黑客可远程控制被感染电脑
4. **下载其他恶意软件**：下载更多病毒或木马

### 1.4 网络钓鱼

#### 什么是网络钓鱼

**网络钓鱼（Phishing）**是指攻击者伪装成可信的网站或机构，诱骗用户输入敏感信息（如账号、密码、银行卡号等）。

#### 常见钓鱼手段

| 手段 | 说明 | 示例 |
|------|------|------|
| 伪造网站 | 制作与真实网站相似的假网站 | 假冒银行网站 |
| 钓鱼邮件 | 发送伪装成正规机构的邮件 | 假冒支付宝邮件 |
| 钓鱼链接 | 通过短信、QQ等发送虚假链接 | "您的账户异常" |
| WiFi钓鱼 | 建立虚假WiFi热点窃取信息 | 免费公共WiFi |

#### 识别钓鱼网站

| 识别方法 | 说明 |
|----------|------|
| 检查网址 | 正规网站网址正确，钓鱼网站网址相似但有差异 |
| 查看证书 | 点击地址栏锁图标查看SSL证书 |
| 注意内容 | 钓鱼网站可能有错别字、排版混乱 |
| 警惕要求 | 正规网站不会要求输入敏感信息 |

**示例**：
- 真实网站：`https://www.taobao.com`
- 钓鱼网站：`https://www.taoba0.com`（数字0代替字母o）

### 2.2 密码安全

#### 密码安全级别

| 级别 | 特点 | 示例 | 安全性 |
|------|------|------|--------|
| 弱密码 | 简单数字、常见单词 | 123456、password | ❌ 极不安全 |
| 中等密码 | 字母+数字，8位以上 | abc12345 | ⚠️ 较不安全 |
| 强密码 | 大小写+数字+符号，8位以上 | Abc@123456 | ✅ 安全 |

#### 设置强密码原则

1. **长度足够**：至少8位，建议12位以上
2. **复杂度高**：包含大小写字母、数字、特殊符号
3. **不使用个人信息**：不用生日、电话、姓名等
4. **不使用常见词汇**：不用password、123456等
5. **定期更换**：建议3-6个月更换一次
6. **不同账号不同密码**：避免一个密码走天下

#### 密码管理建议

| 建议 | 说明 |
|------|------|
| 使用密码管理器 | 如1Password、LastPass等 |
| 开启两步验证 | 增加账号安全性 |
| 不在公共电脑保存密码 | 防止被他人获取 |
| 不告诉他人密码 | 包括亲友和"客服" |

### 2.4 数据安全

#### 数据备份

| 备份方式 | 说明 | 优点 | 缺点 |
|----------|------|------|------|
| 本地备份 | 复制到移动硬盘 | 速度快 | 可能同时损坏 |
| 云端备份 | 上传到云存储 | 异地保存 | 需要网络 |
| 系统还原点 | 创建系统还原点 | 可恢复系统 | 占用空间 |

#### 备份原则

1. **重要数据多处备份**：至少备份到两个不同位置
2. **定期备份**：重要数据每天备份，一般数据每周备份
3. **验证备份**：定期检查备份是否可用
4. **分类备份**：按重要程度分类备份

### 2.5 上网安全习惯

#### 安全上网原则

| 原则 | 说明 |
|------|------|
| 不访问不良网站 | 不浏览暴力、色情、赌博网站 |
| 不下载不明软件 | 从官网或正规应用商店下载 |
| 不点击可疑链接 | 不点击来历不明的链接 |
| 不打开陌生附件 | 不打开陌生人发送的邮件附件 |
| 不连接陌生WiFi | 公共WiFi可能存在风险 |

#### 个人信息保护

| 信息类型 | 保密级别 | 保护措施 |
|----------|----------|----------|
| 身份证号 | ⭐⭐⭐⭐⭐ | 不随意填写，只在正规平台使用 |

## ⚖️ 第三部分：法律法规

### 3.1 网络安全相关法律

#### 主要法律法规

| 法律名称 | 施行时间 | 主要内容 |
|----------|----------|----------|
| 《网络安全法》 | 2017年6月1日 | 网络安全基本法律 |
| 《数据安全法》 | 2021年9月1日 | 数据安全管理 |
| 《个人信息保护法》 | 2021年11月1日 | 个人信息保护 |
| 《未成年人保护法》 | 2021年6月1日 | 未成年人网络保护 |

#### 《网络安全法》要点

| 内容 | 说明 |
|------|------|
| 网络运营者责任 | 应当要求用户提供真实身份信息 |
| 个人信息保护 | 不得泄露、篡改、损毁个人信息 |
| 未成年人保护 | 为未成年人提供网络保护措施 |
| 违法信息处置 | 发现违法信息应及时处置 |

### 3.2 网络行为规范

#### 文明上网规范

| 规范 | 说明 |
|------|------|
| 不传播谣言 | 不转发未经证实的信息 |
| 不网络暴力 | 不进行人身攻击、恶意评论 |
| 不侵犯隐私 | 不公开他人隐私信息 |
| 不传播不良信息 | 不传播暴力、色情、赌博信息 |
| 不传播盗版 | 不传播盗版软件、影视、音乐 |

#### 网络犯罪类型

| 犯罪类型 | 说明 | 法律后果 |
|----------|------|----------|
| 网络诈骗 | 通过网络骗取财物 | 刑事责任 |
| 侵犯公民个人信息 | 非法获取、出售个人信息 | 刑事责任 |
| 网络攻击 | 非法入侵计算机系统 | 刑事责任 |
| 传播病毒 | 故意传播计算机病毒 | 刑事责任 |
| 网络诽谤 | 在网络上捏造事实诽谤他人 | 行政/刑事责任 |

### 3.3 知识产权保护

#### 知识产权类型

| 类型 | 说明 | 保护期限 |
|------|------|----------|
| 著作权（版权） | 文学、艺术、科学作品 | 作者终生+50年 |
| 专利权 | 发明创造 | 发明20年，实用新型10年 |
| 商标权 | 商品或服务标识 | 10年（可续展） |

#### 著作权保护

| 权利类型 | 说明 |
|----------|------|
| 发表权 | 决定是否公开发表 |
| 署名权 | 表明作者身份 |
| 修改权 | 修改或授权他人修改 |
| 复制权 | 制作作品副本 |
| 信息网络传播权 | 在网络上传播作品 |

#### 合理使用

**合理使用**是指在特定情况下使用他人作品，可以不经著作权人许可，不向其支付报酬。

| 合理使用情形 | 说明 |
|--------------|------|
| 个人学习 | 为个人学习、研究使用 |
| 课堂教学 | 为学校课堂教学使用 |
| 新闻报道 | 新闻报道中适当引用 |
| 评论引用 | 为介绍、评论适当引用 |

### 3.4 未成年人网络保护

#### 青少年模式

| 功能 | 说明 |
|------|------|
| 使用时间限制 | 限制每日使用时长 |
| 内容过滤 | 过滤不良信息 |
| 消费限制 | 限制充值、打赏 |
| 交友限制 | 限制陌生人交友 |

#### 防沉迷系统

| 规定 | 说明 |
|------|------|
| 实名认证 | 网络游戏需实名注册 |
| 时间限制 | 未成年人游戏时间受限 |
| 消费限制 | 未成年人充值受限 |

## 🔐 第四部分：实用防护技能

### 4.1 识别网络诈骗

#### 常见诈骗手段

| 诈骗类型 | 手段 | 识别方法 |
|----------|------|----------|
| 冒充客服 | 假冒客服要求退款、验证 | 官方客服不会索要密码 |
| 冒充熟人 | 盗号后借钱 | 电话或视频核实 |
| 虚假中奖 | 通知中奖需交税、手续费 | 正规中奖不收费 |
| 虚假兼职 | 刷单兼职骗取押金 | 天上不会掉馅饼 |
| 虚假投资 | 高回报投资骗局 | 高收益必有高风险 |

#### 防骗原则

1. **不轻信**：不轻信陌生人的话
2. **不透露**：不透露个人信息和密码
3. **不转账**：不向陌生账户转账
4. **多核实**：遇到可疑情况多核实

### 4.2 应对安全事件

#### 发现病毒感染

| 步骤 | 操作 |
|------|------|
| 1 | 立即断开网络连接 |
| 2 | 使用杀毒软件全盘扫描 |
| 3 | 清除或隔离病毒文件 |
| 4 | 修改重要账号密码 |
| 5 | 恢复被感染的文件 |

#### 账号被盗处理

| 步骤 | 操作 |
|------|------|
| 1 | 尝试通过密码找回功能找回账号 |
| 2 | 联系客服进行账号冻结 |
| 3 | 修改关联账号的密码 |
| 4 | 检查账号是否有异常操作 |
| 5 | 开启两步验证等安全措施 |

### 四、简答题

**1. 计算机病毒的特点：**

1. **传染性**：能自我复制并传播到其他文件或系统
2. **破坏性**：破坏数据、占用资源、导致系统崩溃
3. **隐蔽性**：伪装成正常程序，不易被发现
4. **潜伏性**：感染后可能不立即发作，等待触发条件
5. **可触发性**：满足特定条件时激活，如特定日期

**2. 设置安全密码的原则：**

1. **长度足够**：至少8位，建议12位以上
2. **复杂度高**：包含大小写字母、数字、特殊符号
3. **不使用个人信息**：不用生日、电话、姓名等
4. **不使用常见词汇**：不用password、123456等
5. **定期更换**：建议3-6个月更换一次
6. **不同账号不同密码**：避免一个密码走天下

**3. 文明上网的做法：**

1. 不在网络上散布谣言，转发信息前先核实
2. 不进行网络暴力，不恶意评论、人身攻击
3. 不传播不良信息，包括暴力、色情、赌博内容
4. 尊重他人隐私，不公开他人隐私信息
5. 尊重知识产权，不传播盗版软件、影视、音乐
6. 保护个人信息，不随意透露自己的信息
7. 使用文明用语，不使用脏话、攻击性语言
8. 遇到不良信息及时举报
