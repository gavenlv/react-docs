## 🌐 第一部分：计算机网络概述

### 1.1 什么是计算机网络

**计算机网络**是将地理位置不同的计算机，通过通信线路连接起来，在网络软件的管理下，实现资源共享和信息传递的系统。

#### 网络的组成

| 组成部分 | 说明 |
|----------|------|
| 计算机 | 网络中的终端设备 |
| 通信线路 | 连接设备的传输介质 |
| 网络设备 | 路由器、交换机等 |
| 网络软件 | 操作系统、协议等 |

### 1.2 网络的功能

| 功能 | 说明 | 例子 |
|------|------|------|
| 资源共享 | 共享硬件、软件、数据 | 打印机共享、文件共享 |
| 信息传递 | 传输各种信息 | 电子邮件、即时通讯 |
| 分布式处理 | 多台计算机协同工作 | 云计算、分布式计算 |
| 提高可靠性 | 互为备份，提高可靠性 | 服务器集群 |

### 1.3 网络的发展

| 阶段 | 特点 |
|------|------|
| 第一代 | 远程终端联机阶段 |
| 第二代 | 计算机网络阶段 |
| 第三代 | 计算机网络互联阶段 |
| 第四代 | 信息高速公路阶段 |

## 📡 第二部分：网络分类

### 2.1 按覆盖范围分类

| 类型 | 英文 | 范围 | 举例 |
|------|------|------|------|
| 局域网 | LAN | 几百米~几公里 | 学校、家庭、办公室 |
| 城域网 | MAN | 几十公里 | 城市 |
| 广域网 | WAN | 跨地区、跨国 | 互联网 |

#### 局域网（LAN）

- **特点**：覆盖范围小、传输速率高、误码率低
- **应用**：学校机房、家庭网络、办公室网络
- **技术**：以太网、Wi-Fi

#### 城域网（MAN）

- **特点**：覆盖一个城市
- **应用**：城市政务网、教育城域网
- **技术**：光纤传输

#### 广域网（WAN）

- **特点**：覆盖范围大、传输速率较低
- **应用**：互联网、跨国公司网络
- **技术**：光纤、卫星通信

### 2.2 按拓扑结构分类

#### 总线型

```
    ┌───┐   ┌───┐   ┌───┐   ┌───┐
    │ A │───│ B │───│ C │───│ D │
    └───┘   └───┘   └───┘   └───┘
        └─────────────────────┘
              总线（主干线）
```

- **特点**：结构简单、成本低、扩展方便
- **缺点**：故障检测困难、总线故障影响全网

#### 星型

```
            ┌───┐
            │ A │
            └───┘
              │
    ┌───┐   ┌───┐   ┌───┐
    │ B │───│中心│───│ C │
    └───┘   └───┘   └───┘
              │
            ┌───┐
            │ D │
            └───┘
```

- **特点**：易于扩展、故障隔离好
- **缺点**：中心故障影响全网、成本较高

#### 环型

```
        ┌───┐
    ┌───│ A │───┐
    │   └───┘   │
┌───┐         ┌───┐
│ D │         │ B │
└───┘         └───┘
    │   ┌───┐   │
    └───│ C │───┘
        └───┘
```

- **特点**：数据单向传输、无冲突
- **缺点**：节点故障影响全网、扩展困难

#### 网状型

```
    ┌───┐───────┌───┐
    │ A │───────│ B │
    └───┘\     /└───┘
      │   \ /   │
      │   / \   │
    ┌───┘/     \┌───┐
    │ D │───────│ C │
    └───┘       └───┘
```

- **特点**：可靠性高、多条路径
- **缺点**：结构复杂、成本高

### 2.3 按传输介质分类

| 类型 | 介质 | 特点 |
|------|------|------|
| 有线网 | 双绞线、光纤、同轴电缆 | 稳定、速度快 |
| 无线网 | 无线电波、微波、红外线 | 灵活、移动方便 |

## 🔢 第三部分：IP 地址

### 3.1 什么是 IP 地址

**IP 地址**是网络中设备的唯一标识，类似于手机的电话号码。

#### IP 地址的作用

- 标识网络中的设备
- 实现设备之间的通信
- 路由寻址

### 3.2 IPv4 地址

#### 格式

- 32 位二进制数
- 点分十进制表示：`xxx.xxx.xxx.xxx`
- 每段范围：0-255
- 示例：`192.168.1.1`

#### IPv4 地址结构

```
网络号（Network ID）+ 主机号（Host ID）

示例：192.168.1.1
      ├─────┤ ├──┤
      网络号   主机号
```

#### IP 地址分类

| 类别 | 范围 | 网络数 | 主机数 | 用途 |
|------|------|--------|--------|------|
| A类 | 1.0.0.0 - 126.255.255.255 | 126 | 1677万 | 大型网络 |
| B类 | 128.0.0.0 - 191.255.255.255 | 16384 | 65534 | 中型网络 |
| C类 | 192.0.0.0 - 223.255.255.255 | 209万 | 254 | 小型网络 |
| D类 | 224.0.0.0 - 239.255.255.255 | - | - | 组播地址 |
| E类 | 240.0.0.0 - 255.255.255.255 | - | - | 保留地址 |

### 3.3 特殊 IP 地址

| 地址 | 用途 |
|------|------|
| 127.0.0.1 | 本机回环地址（localhost） |
| 0.0.0.0 | 当前网络 |
| 255.255.255.255 | 广播地址 |
| 192.168.x.x | 私有地址（局域网） |
| 10.x.x.x | 私有地址（局域网） |
| 172.16.x.x - 172.31.x.x | 私有地址（局域网） |

### 3.4 子网掩码

**子网掩码**用于区分网络号和主机号。

| 类别 | 默认子网掩码 |
|------|--------------|
| A类 | 255.0.0.0 |
| B类 | 255.255.0.0 |
| C类 | 255.255.255.0 |

### 3.5 IPv6 地址

#### 为什么需要 IPv6

- IPv4 地址资源枯竭
- IPv4 只有约 43 亿个地址
- IPv6 有 2^128 个地址

#### IPv6 格式

- 128 位二进制数
- 冒号十六进制表示
- 示例：`2001:0db8:85a3:0000:0000:8a2e:0370:7334`

## 📛 第四部分：域名系统

### 4.1 什么是域名

**域名**是 IP 地址的别名，便于记忆和使用。

#### 为什么需要域名

- IP 地址难以记忆
- 域名有意义、易记忆
- 域名可以变更 IP 地址

### 4.2 域名结构

```
        www.baidu.com.cn
        │    │     │  │
        │    │     │  └── 顶级域名（国家域名）
        │    │     └───── 二级域名
        │    └─────────── 三级域名
        └──────────────── 主机名
```

#### 域名从右到左依次为：

1. **顶级域名**：最高级别的域名
2. **二级域名**：顶级域名下的域名
3. **三级域名**：二级域名下的域名
4. **主机名**：具体的服务器名称

### 4.3 常见顶级域名

#### 机构域名

| 域名 | 含义 |
|------|------|
| .com | 商业机构 |
| .edu | 教育机构 |
| .gov | 政府机构 |
| .org | 非营利组织 |
| .net | 网络服务机构 |
| .mil | 军事机构 |

#### 国家/地区域名

| 域名 | 国家/地区 |
|------|-----------|
| .cn | 中国 |
| .us | 美国 |
| .jp | 日本 |
| .uk | 英国 |
| .hk | 香港 |
| .tw | 台湾 |

### 4.4 DNS 服务

**DNS（Domain Name System）** 是域名系统，负责将域名解析为 IP 地址。

#### DNS 解析过程

```
用户输入域名 → DNS 服务器查询 → 返回 IP 地址 → 访问网站
```

#### DNS 服务器

- 存储域名和 IP 地址的对应关系
- 提供域名解析服务
- 常用 DNS：114.114.114.114、8.8.8.8

## 🔌 第五部分：网络设备与协议

### 5.1 常见网络设备

| 设备 | 功能 | 工作层次 |
|------|------|----------|
| 网卡 | 连接计算机与网络 | 物理层 |
| 集线器（Hub） | 多端口中继器 | 物理层 |
| 交换机 | 数据交换 | 数据链路层 |
| 路由器 | 路由选择、网络互联 | 网络层 |
| 调制解调器 | 信号转换 | 物理层 |

### 5.2 网络协议

#### 什么是网络协议

**网络协议**是计算机网络中进行数据交换而建立的规则、标准或约定的集合。

#### 常见协议

| 协议 | 全称 | 用途 |
|------|------|------|
| HTTP | 超文本传输协议 | 网页浏览 |
| HTTPS | 安全超文本传输协议 | 加密网页浏览 |
| FTP | 文件传输协议 | 文件传输 |
| SMTP | 简单邮件传输协议 | 发送邮件 |
| POP3 | 邮局协议第3版 | 接收邮件 |
| TCP | 传输控制协议 | 可靠传输 |
| UDP | 用户数据报协议 | 快速传输 |
| IP | 网际协议 | 网络寻址 |

### 5.3 TCP/IP 协议模型

| 层次 | 协议 |
|------|------|
| 应用层 | HTTP、FTP、SMTP、DNS |
| 传输层 | TCP、UDP |
| 网络层 | IP、ICMP |
| 网络接口层 | 以太网、Wi-Fi |

### 四、简答题

**1. 局域网和广域网的主要区别：**

| 对比项 | 局域网（LAN） | 广域网（WAN） |
|--------|---------------|---------------|
| 覆盖范围 | 几百米到几公里 | 跨地区、跨国 |
| 传输速率 | 高 | 较低 |

**3. IP 地址解释：**

- 属于 C 类地址（范围 192.0.0.0 - 223.255.255.255）
- 是私有 IP 地址（192.168.x.x）
