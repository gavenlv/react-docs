## 📖 第一部分：Word 概述

### 1.1 Word 简介

**Microsoft Word** 是微软公司开发的文字处理软件，是目前最流行的文字处理工具之一。

#### Word 的主要功能

| 功能 | 说明 |
|------|------|
| 文字编辑 | 输入、修改、删除文字 |
| 格式设置 | 字体、段落、页面格式 |
| 表格制作 | 创建和编辑表格 |
| 图文混排 | 插入图片、图形 |
| 文档打印 | 打印设置和输出 |

#### Word 文件格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| Word 文档 | .docx | 默认格式 |
| Word 97-2003 | .doc | 旧版本格式 |
| PDF 文档 | .pdf | 可导出为PDF |
| RTF 文档 | .rtf | 富文本格式 |

### 1.2 Word 界面组成

```
┌─────────────────────────────────────────────────────────┐
│  标题栏（文件名 + 控制按钮）                              │
├─────────────────────────────────────────────────────────┤
│  功能区（选项卡 + 功能组）                               │
│  ┌────┬────┬────┬────┬────┐                            │
│  │开始│插入│设计│布局│引用│                            │
│  └────┴────┴────┴────┴────┘                            │
├─────────────────────────────────────────────────────────┤
│  标尺                                                   │
├─────────────────────────────────────────────────────────┤
│                                                         │
│                    文档编辑区                           │
│                                                         │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  状态栏                    │  视图按钮  │  缩放滑块     │
└─────────────────────────────────────────────────────────┘
```

#### 界面元素说明

| 元素 | 功能 |
|------|------|
| **标题栏** | 显示文件名，包含控制按钮 |
| **功能区** | 包含所有命令和工具 |
| **选项卡** | 功能分类标签（开始、插入等） |
| **功能组** | 每个选项卡下的功能分组 |
| **标尺** | 显示页边距和缩进位置 |
| **编辑区** | 输入和编辑文档内容 |
| **状态栏** | 显示文档状态信息 |
| **视图按钮** | 切换文档视图方式 |

### 1.3 视图方式

| 视图 | 说明 | 用途 |
|------|------|------|
| **页面视图** | 显示实际打印效果 | 默认视图，适合编辑 |
| **阅读视图** | 优化阅读体验 | 阅读文档 |
| **Web版式视图** | 显示网页效果 | 制作网页文档 |
| **大纲视图** | 显示文档结构 | 整理文档结构 |
| **草稿视图** | 简化显示 | 快速编辑 |

## ✂️ 第三部分：文档编辑

### 3.1 选择文本

#### 基本选择方法

| 选择对象 | 操作方法 |
|----------|----------|
| 任意文本 | 拖动鼠标选择 |
| 一个词 | 双击该词 |
| 一句话 | Ctrl + 点击句子任意位置 |
| 一行 | 点击行左侧空白处 |
| 多行 | 在左侧空白处拖动 |
| 一个段落 | 三击段落 或 双击左侧空白处 |
| 整篇文档 | Ctrl + A 或 三击左侧空白处 |

#### 使用键盘选择

| 快捷键 | 功能 |
|--------|------|
| Shift + → | 向右选择一个字符 |
| Shift + ← | 向左选择一个字符 |
| Shift + ↓ | 向下选择一行 |
| Shift + ↑ | 向上选择一行 |
| Ctrl + Shift + → | 向右选择一个词 |
| Ctrl + Shift + ← | 向左选择一个词 |
| Ctrl + Shift + End | 选择到文档末尾 |
| Ctrl + Shift + Home | 选择到文档开头 |

### 3.2 复制、剪切与粘贴

#### 操作说明

| 操作 | 说明 | 快捷键 |
|------|------|--------|
| 复制 | 复制选中的内容到剪贴板，原位置保留 | Ctrl + C |
| 剪切 | 移动选中的内容到剪贴板，原位置删除 | Ctrl + X |
| 粘贴 | 将剪贴板内容插入到当前位置 | Ctrl + V |

#### 粘贴选项

| 选项 | 说明 |
|------|------|
| 保留源格式 | 保持原内容的格式 |
| 合并格式 | 合并原格式和当前位置格式 |
| 只保留文本 | 只粘贴文字，去除所有格式 |
| 图片 | 粘贴为图片 |

#### 操作步骤

**复制粘贴：**
1. 选中要复制的内容
2. 按 Ctrl + C（或右键 → 复制）
3. 定位到目标位置
4. 按 Ctrl + V（或右键 → 粘贴）

**剪切粘贴（移动）：**
1. 选中要移动的内容
2. 按 Ctrl + X（或右键 → 剪切）
3. 定位到目标位置
4. 按 Ctrl + V（或右键 → 粘贴）

### 3.3 撤销与重做

| 操作 | 说明 | 快捷键 |
|------|------|--------|
| 撤销 | 取消上一步操作 | Ctrl + Z |
| 重做 | 恢复被撤销的操作 | Ctrl + Y |
| 重复 | 重复上一步操作 | Ctrl + Y |

#### 多次撤销

- 点击快速访问工具栏的撤销按钮旁边的箭头
- 选择要撤销的操作
- 可以一次撤销多步操作

### 3.4 查找与替换

#### 查找功能

**用途**：在文档中快速定位特定文字。

**操作方法**：
1. 按 Ctrl + F
2. 在搜索框中输入要查找的内容
3. 点击"查找下一处"

#### 替换功能

**用途**：将文档中的某些文字替换为其他文字。

**操作方法**：
1. 按 Ctrl + H（或 开始 → 替换）
2. 在"查找内容"框中输入要查找的文字
3. 在"替换为"框中输入替换后的文字
4. 点击"替换"或"全部替换"

#### 高级查找替换

| 功能 | 说明 |
|------|------|
| 区分大小写 | 区分字母大小写 |
| 全字匹配 | 只匹配完整的单词 |
| 使用通配符 | 使用 * 和 ? 等通配符 |
| 格式替换 | 查找或替换特定格式的文字 |
| 特殊格式 | 查找段落标记、制表符等特殊字符 |

#### 替换实例

**实例1：批量替换文字**
- 查找内容：电脑
- 替换为：计算机
- 点击"全部替换"

**实例2：批量删除空行**
- 查找内容：^p^p（两个段落标记）
- 替换为：^p（一个段落标记）
- 点击"全部替换"

### 3.5 移动与复制文本

#### 拖动移动

1. 选中要移动的文本
2. 按住鼠标左键拖动到目标位置
3. 释放鼠标

#### 拖动复制

1. 选中要复制的文本
2. 按住 Ctrl 键的同时拖动到目标位置
3. 先释放鼠标，再释放 Ctrl 键

### 3.6 删除文本

| 操作 | 方法 |
|------|------|
| 删除光标后字符 | Delete 键 |
| 删除光标前字符 | Backspace 键 |
| 删除选中文本 | Delete 或 Backspace |
| 删除一个词 | Ctrl + Delete 或 Ctrl + Backspace |

### 3.7 文本定位

| 快捷键 | 功能 |
|--------|------|
| Home | 移到行首 |
| End | 移到行尾 |
| Ctrl + Home | 移到文档开头 |
| Ctrl + End | 移到文档末尾 |
| Ctrl + → | 向右移动一个词 |
| Ctrl + ← | 向左移动一个词 |
| Page Up | 向上翻页 |
| Page Down | 向下翻页 |

## 📐 第四部分：段落格式

### 4.1 对齐方式

| 对齐方式 | 快捷键 | 说明 |
|----------|--------|------|
| 左对齐 | Ctrl + L | 文字靠左对齐 |
| 居中 | Ctrl + E | 文字居中对齐 |
| 右对齐 | Ctrl + R | 文字靠右对齐 |
| 两端对齐 | Ctrl + J | 文字左右两端都对齐 |
| 分散对齐 | - | 文字均匀分布 |

### 4.2 缩进设置

#### 缩进类型

| 类型 | 说明 | 设置位置 |
|------|------|----------|
| 首行缩进 | 段落第一行缩进 | 开始 → 段落 → 缩进 |
| 悬挂缩进 | 除第一行外其他行缩进 | 开始 → 段落 → 缩进 |
| 左缩进 | 整个段落左边缩进 | 标尺左侧滑块 |
| 右缩进 | 整个段落右边缩进 | 标尺右侧滑块 |

#### 设置首行缩进

**常用设置**：首行缩进 2 字符

1. 选中段落
2. 右键 → 段落
3. 缩进 → 特殊格式 → 首行缩进
4. 设置值为 2 字符

### 4.3 行距设置

| 行距 | 快捷键 | 说明 |
|------|--------|------|
| 单倍行距 | Ctrl + 1 | 默认行距 |
| 1.5 倍行距 | Ctrl + 5 | 常用行距 |
| 2 倍行距 | Ctrl + 2 | 较大行距 |

#### 行距设置方法

1. 选中段落
2. 右键 → 段落
3. 行距 → 选择行距类型

### 4.4 段落间距

| 设置 | 说明 |
|------|------|
| 段前 | 段落前的空白距离 |
| 段后 | 段落后的空白距离 |

#### 设置方法

1. 选中段落
2. 右键 → 段落
3. 间距 → 段前/段后 → 输入数值

### 4.5 项目符号和编号

#### 项目符号

为段落添加符号标记。

**设置方法**：开始 → 项目符号 → 选择符号样式

#### 编号

为段落添加数字编号。

**设置方法**：开始 → 编号 → 选择编号样式

#### 多级列表

创建多级层次的列表。

**设置方法**：开始 → 多级列表 → 选择列表样式

## 📏 第五部分：页面设置

### 5.1 页边距

**页边距**是页面内容与纸张边缘的距离。

| 类型 | 用途 |
|------|------|
| 普通 | 默认设置，上下2.54cm，左右3.17cm |
| 窄 | 较小页边距，节省纸张 |
| 适中 | 上下左右均为2.54cm |
| 宽 | 较大页边距，适合装订 |

#### 设置方法

布局 → 页边距 → 选择预设或自定义

### 5.2 纸张方向

| 方向 | 说明 |
|------|------|
| 纵向 | 纸张竖放（默认） |
| 横向 | 纸张横放 |

#### 设置方法

布局 → 纸张方向 → 选择方向

### 5.3 纸张大小

| 纸张 | 尺寸 |
|------|------|
| A4 | 210mm × 297mm（默认） |
| A3 | 297mm × 420mm |
| B5 | 176mm × 250mm |
| Letter | 216mm × 279mm |

#### 设置方法

布局 → 纸张大小 → 选择纸张
