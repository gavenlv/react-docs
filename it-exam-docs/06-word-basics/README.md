## 📖 第一部分：Word 概述

### 1.1 Word 简介

**Microsoft Word** 是微软公司开发的文字处理软件，是目前最流行的文字处理工具之一。

#### Word 的主要功能

| 功能 | 说明 |
|------|------|
| 文字编辑 | 输入、修改、删除文字 |
| 格式设置 | 字体、段落、页面格式 |
| 表格制作 | 创建和编辑表格 |
| 图文混排 | 插入图片、图形 |
| 文档打印 | 打印设置和输出 |

#### Word 文件格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| Word 文档 | .docx | 默认格式 |
| Word 97-2003 | .doc | 旧版本格式 |
| PDF 文档 | .pdf | 可导出为PDF |
| RTF 文档 | .rtf | 富文本格式 |

### 1.2 Word 界面组成

```
┌─────────────────────────────────────────────────────────┐
│  标题栏（文件名 + 控制按钮）                              │
├─────────────────────────────────────────────────────────┤
│  功能区（选项卡 + 功能组）                               │
│  ┌────┬────┬────┬────┬────┐                            │
│  │开始│插入│设计│布局│引用│                            │
│  └────┴────┴────┴────┴────┘                            │
├─────────────────────────────────────────────────────────┤
│  标尺                                                   │
├─────────────────────────────────────────────────────────┤
│                                                         │
│                    文档编辑区                           │
│                                                         │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  状态栏                    │  视图按钮  │  缩放滑块     │
└─────────────────────────────────────────────────────────┘
```

#### 界面元素说明

| 元素 | 功能 |
|------|------|
| **标题栏** | 显示文件名，包含控制按钮 |
| **功能区** | 包含所有命令和工具 |
| **选项卡** | 功能分类标签（开始、插入等） |
| **功能组** | 每个选项卡下的功能分组 |
| **标尺** | 显示页边距和缩进位置 |
| **编辑区** | 输入和编辑文档内容 |
| **状态栏** | 显示文档状态信息 |
| **视图按钮** | 切换文档视图方式 |

### 1.3 视图方式

| 视图 | 说明 | 用途 |
|------|------|------|
| **页面视图** | 显示实际打印效果 | 默认视图，适合编辑 |
| **阅读视图** | 优化阅读体验 | 阅读文档 |
| **Web版式视图** | 显示网页效果 | 制作网页文档 |
| **大纲视图** | 显示文档结构 | 整理文档结构 |
| **草稿视图** | 简化显示 | 快速编辑 |

## 📄 第二部分：文档基本操作

### 2.1 新建文档

| 方法 | 操作步骤 |
|------|----------|
| 方法一 | 文件 → 新建 → 空白文档 |
| 方法二 | Ctrl + N |
| 方法三 | 右键桌面 → 新建 → Microsoft Word 文档 |

### 2.2 打开文档

| 方法 | 操作步骤 |
|------|----------|
| 方法一 | 文件 → 打开 → 选择文件 |
| 方法二 | Ctrl + O |
| 方法三 | 双击 Word 文档文件 |
| 方法四 | 最近使用的文档列表 |

### 2.3 保存文档

| 方法 | 操作步骤 |
|------|----------|
| 保存 | Ctrl + S |
| 另存为 | 文件 → 另存为 → 选择位置和格式 |

#### 保存注意事项

- 首次保存会弹出"另存为"对话框
- 建议设置自动保存（文件 → 选项 → 保存）
- 养成经常保存的习惯

### 2.4 关闭文档

| 方法 | 操作步骤 |
|------|----------|
| 方法一 | 文件 → 关闭 |
| 方法二 | Ctrl + W |
| 方法三 | 点击文档窗口的关闭按钮 |

## 🔤 第三部分：文字格式

### 3.1 字体设置

#### 设置方法

| 方法 | 操作步骤 |
|------|----------|
| 功能区 | 开始 → 字体组 |
| 对话框 | 右键 → 字体 或 Ctrl + D |
| 浮动工具栏 | 选中文字后出现 |

#### 字体设置选项

| 设置项 | 说明 | 快捷键 |
|--------|------|--------|
| 字体 | 设置字体类型 | - |
| 字号 | 设置文字大小 | - |
| 加粗 | 文字加粗 | Ctrl + B |
| 斜体 | 文字倾斜 | Ctrl + I |
| 下划线 | 添加下划线 | Ctrl + U |
| 删除线 | 添加删除线 | - |
| 字体颜色 | 设置文字颜色 | - |
| 突出显示 | 文字背景高亮 | - |

### 3.2 字号对照表

| 字号 | 磅值 | 用途 |
|------|------|------|
| 初号 | 42 | 大标题 |
| 小初 | 36 | 大标题 |
| 一号 | 26 | 标题 |
| 小一 | 24 | 标题 |
| 二号 | 22 | 标题 |
| 小二 | 18 | 标题 |
| 三号 | 16 | 正文/小标题 |
| 小三 | 15 | 正文/小标题 |
| 四号 | 14 | 正文 |
| 小四 | 12 | 正文（常用） |
| 五号 | 10.5 | 正文 |
| 小五 | 9 | 注释 |

### 3.3 字体效果

| 效果 | 说明 |
|------|------|
| 上标 | 文字缩小并上移，如 x² |
| 下标 | 文字缩小并下移，如 H₂O |
| 小型大写字母 | 小写字母显示为大写但字号较小 |
| 全部大写 | 所有字母显示为大写 |

## 📐 第四部分：段落格式

### 4.1 对齐方式

| 对齐方式 | 快捷键 | 说明 |
|----------|--------|------|
| 左对齐 | Ctrl + L | 文字靠左对齐 |
| 居中 | Ctrl + E | 文字居中对齐 |
| 右对齐 | Ctrl + R | 文字靠右对齐 |
| 两端对齐 | Ctrl + J | 文字左右两端都对齐 |
| 分散对齐 | - | 文字均匀分布 |

### 4.2 缩进设置

#### 缩进类型

| 类型 | 说明 | 设置位置 |
|------|------|----------|
| 首行缩进 | 段落第一行缩进 | 开始 → 段落 → 缩进 |
| 悬挂缩进 | 除第一行外其他行缩进 | 开始 → 段落 → 缩进 |
| 左缩进 | 整个段落左边缩进 | 标尺左侧滑块 |
| 右缩进 | 整个段落右边缩进 | 标尺右侧滑块 |

#### 设置首行缩进

**常用设置**：首行缩进 2 字符

1. 选中段落
2. 右键 → 段落
3. 缩进 → 特殊格式 → 首行缩进
4. 设置值为 2 字符

### 4.3 行距设置

| 行距 | 快捷键 | 说明 |
|------|--------|------|
| 单倍行距 | Ctrl + 1 | 默认行距 |
| 1.5 倍行距 | Ctrl + 5 | 常用行距 |
| 2 倍行距 | Ctrl + 2 | 较大行距 |

#### 行距设置方法

1. 选中段落
2. 右键 → 段落
3. 行距 → 选择行距类型

### 4.4 段落间距

| 设置 | 说明 |
|------|------|
| 段前 | 段落前的空白距离 |
| 段后 | 段落后的空白距离 |

#### 设置方法

1. 选中段落
2. 右键 → 段落
3. 间距 → 段前/段后 → 输入数值

### 4.5 项目符号和编号

#### 项目符号

为段落添加符号标记。

**设置方法**：开始 → 项目符号 → 选择符号样式

#### 编号

为段落添加数字编号。

**设置方法**：开始 → 编号 → 选择编号样式

#### 多级列表

创建多级层次的列表。

**设置方法**：开始 → 多级列表 → 选择列表样式

## 📏 第五部分：页面设置

### 5.1 页边距

**页边距**是页面内容与纸张边缘的距离。

| 类型 | 用途 |
|------|------|
| 普通 | 默认设置，上下2.54cm，左右3.17cm |
| 窄 | 较小页边距，节省纸张 |
| 适中 | 上下左右均为2.54cm |
| 宽 | 较大页边距，适合装订 |

#### 设置方法

布局 → 页边距 → 选择预设或自定义

### 5.2 纸张方向

| 方向 | 说明 |
|------|------|
| 纵向 | 纸张竖放（默认） |
| 横向 | 纸张横放 |

#### 设置方法

布局 → 纸张方向 → 选择方向

### 5.3 纸张大小

| 纸张 | 尺寸 |
|------|------|
| A4 | 210mm × 297mm（默认） |
| A3 | 297mm × 420mm |
| B5 | 176mm × 250mm |
| Letter | 216mm × 279mm |

#### 设置方法

布局 → 纸张大小 → 选择纸张

### 四、操作题

**1. 操作步骤：**
- 新建 Word 文档
- 输入文字
- 选中文字
- 开始 → 字体 → 宋体
- 开始 → 字号 → 四号
- 开始 → 加粗（或 Ctrl + B）

**2. 操作步骤：**
- 选中段落
- 右键 → 段落
- 缩进 → 特殊格式 → 首行缩进 → 2字符
- 行距 → 1.5 倍行距
- 间距 → 段前 → 0.5 行

**3. 操作步骤：**
- 布局 → 纸张大小 → A4
- 布局 → 纸张方向 → 纵向
- 布局 → 页边距 → 普通
