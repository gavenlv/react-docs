## 📐 第一部分：公式基础

### 1.1 公式概念

**公式**是以等号 `=` 开头，由运算符、常量、单元格引用、函数等组成的表达式，用于计算并返回结果。

#### 公式的组成

| 组成部分 | 说明 | 示例 |
|----------|------|------|
| 等号 | 公式必须以等号开头 | = |
| 常量 | 固定的数值或文本 | 100, "及格" |
| 单元格引用 | 引用其他单元格的值 | A1, B2:C5 |
| 运算符 | 进行计算的符号 | +, -, *, / |
| 函数 | 预定义的计算公式 | SUM(), AVERAGE() |

### 1.2 输入公式

#### 基本步骤

1. 选中要输入公式的单元格
2. 输入等号 `=`
3. 输入公式内容
4. 按 Enter 确认

#### 示例

```
=A1+B1        计算 A1 和 B1 的和
=A1*10        计算 A1 乘以 10
=(A1+B1)/2    计算 A1 和 B1 的平均值
=A1&"分"      将 A1 的值与"分"连接
```

### 1.3 运算符

#### 算术运算符

| 运算符 | 功能 | 示例 | 结果 |
|--------|------|------|------|
| + | 加法 | =5+3 | 8 |
| - | 减法 | =5-3 | 2 |
| * | 乘法 | =5*3 | 15 |
| / | 除法 | =6/2 | 3 |
| % | 百分比 | =50% | 0.5 |
| ^ | 乘方 | =2^3 | 8 |

#### 比较运算符

| 运算符 | 功能 | 示例 | 结果 |
|--------|------|------|------|
| = | 等于 | =5=3 | FALSE |
| <> | 不等于 | =5<>3 | TRUE |
| > | 大于 | =5>3 | TRUE |
| < | 小于 | =5<3 | FALSE |
| >= | 大于等于 | =5>=5 | TRUE |
| <= | 小于等于 | =5<=3 | FALSE |

#### 文本运算符

| 运算符 | 功能 | 示例 | 结果 |
|--------|------|------|------|
| & | 连接文本 | ="Hello"&"World" | HelloWorld |
| & | 连接文本和数字 | ="成绩"&90 | 成绩90 |

#### 运算符优先级

| 优先级 | 运算符 | 说明 |
|--------|--------|------|
| 1 | () | 括号（最高优先级） |
| 2 | % | 百分比 |
| 3 | ^ | 乘方 |
| 4 | * / | 乘除 |
| 5 | + - | 加减 |
| 6 | & | 连接 |
| 7 | = <> > < >= <= | 比较 |

### 1.4 编辑公式

#### 修改公式

| 方法 | 操作 |
|------|------|
| 双击单元格 | 直接在单元格中编辑 |
| 按 F2 | 进入编辑模式 |
| 点击编辑栏 | 在编辑栏中修改 |

#### 复制公式

- 选中单元格 → Ctrl + C → 选择目标 → Ctrl + V
- 拖动填充柄复制公式

## 📍 第二部分：单元格引用

### 2.1 引用类型

#### 相对引用

- 格式：A1、B2
- 特点：复制公式时，引用会随位置变化
- 示例：公式 `=A1+B1` 从 C1 复制到 C2，变为 `=A2+B2`

#### 绝对引用

- 格式：$A$1、$B$2
- 特点：复制公式时，引用保持不变
- 示例：公式 `=$A$1+$B$1` 复制到任何位置都不变

#### 混合引用

| 类型 | 格式 | 说明 |
|------|------|------|
| 列绝对、行相对 | $A1 | 列固定，行变化 |
| 列相对、行绝对 | A$1 | 列变化，行固定 |

### 2.2 引用切换

按 F4 键可以在引用类型之间切换：

```
A1 → $A$1 → A$1 → $A1 → A1
```

### 2.3 引用示例

#### 相对引用示例

| 单元格 | 公式 | 复制到 C2 后 |
|--------|------|--------------|
| C1 | =A1+B1 | =A2+B2 |

#### 绝对引用示例

| 单元格 | 公式 | 复制到 C2 后 |
|--------|------|--------------|
| C1 | =$A$1+$B$1 | =$A$1+$B$1 |

#### 混合引用示例

制作九九乘法表：

```
   A    B    C    D    E
1      1    2    3    4
2  1   =$A2*B$1
3  2   =$A3*B$1
4  3   =$A4*B$1
```

### 2.4 跨工作表引用

#### 引用格式

```
=工作表名!单元格地址

示例：
=Sheet2!A1        引用 Sheet2 的 A1 单元格
=成绩表!B5        引用"成绩表"工作表的 B5 单元格
```

#### 跨工作簿引用

```
=[工作簿名]工作表名!单元格地址

示例：
=[成绩.xlsx]Sheet1!A1
```

## 📊 第三部分：常用函数

### 3.1 函数基础

#### 函数结构

```
=函数名(参数1, 参数2, ...)

示例：
=SUM(A1:A10)
=AVERAGE(B1:B5)
=IF(A1>60, "及格", "不及格")
```

#### 输入函数

| 方法 | 操作 |
|------|------|
| 直接输入 | 输入 = 函数名(参数) |
| 函数向导 | 点击 fx 按钮 或 Shift+F3 |
| 自动求和 | 点击【开始】→【自动求和】 |

### 3.2 求和函数 SUM

#### 功能

计算一组数值的总和。

#### 语法

```
=SUM(数值1, 数值2, ...)
=SUM(区域)
```

#### 示例

| 公式 | 说明 |
|------|------|
| =SUM(A1:A10) | 计算 A1 到 A10 的和 |
| =SUM(A1,B1,C1) | 计算 A1、B1、C1 的和 |
| =SUM(A1:A10,C1:C10) | 计算两个区域的总和 |
| =SUM(A1:A10,100) | 计算区域和与 100 的和 |

### 3.3 平均值函数 AVERAGE

#### 功能

计算一组数值的算术平均值。

#### 语法

```
=AVERAGE(数值1, 数值2, ...)
=AVERAGE(区域)
```

#### 示例

| 公式 | 说明 |
|------|------|
| =AVERAGE(A1:A10) | 计算 A1 到 A10 的平均值 |
| =AVERAGE(B2:D2) | 计算 B2 到 D2 的平均值 |

### 3.4 最大值函数 MAX

#### 功能

返回一组数值中的最大值。

#### 语法

```
=MAX(数值1, 数值2, ...)
=MAX(区域)
```

#### 示例

| 公式 | 说明 |
|------|------|
| =MAX(A1:A10) | 返回 A1 到 A10 中的最大值 |
| =MAX(B2:D2) | 返回 B2 到 D2 中的最大值 |

### 3.5 最小值函数 MIN

#### 功能

返回一组数值中的最小值。

#### 语法

```
=MIN(数值1, 数值2, ...)
=MIN(区域)
```

#### 示例

| 公式 | 说明 |
|------|------|
| =MIN(A1:A10) | 返回 A1 到 A10 中的最小值 |
| =MIN(B2:D2) | 返回 B2 到 D2 中的最小值 |

### 3.6 计数函数

#### COUNT 函数

计算包含数值的单元格个数。

```
=COUNT(A1:A10)    统计 A1:A10 中数值单元格的个数
```

#### COUNTA 函数

计算非空单元格的个数。

```
=COUNTA(A1:A10)   统计 A1:A10 中非空单元格的个数
```

#### COUNTIF 函数

按条件计数。

```
=COUNTIF(区域, 条件)

示例：
=COUNTIF(A1:A10, ">=60")     统计大于等于 60 的个数
=COUNTIF(B1:B10, "男")       统计"男"的个数
=COUNTIF(C1:C10, "及格")     统计"及格"的个数
```

### 3.7 条件函数 IF

#### 功能

根据条件返回不同的值。

#### 语法

```
=IF(条件, 条件为真时的值, 条件为假时的值)
```

#### 示例

| 公式 | 说明 |
|------|------|
| =IF(A1>=60, "及格", "不及格") | A1 大于等于 60 显示"及格"，否则显示"不及格" |
| =IF(B1>90, "优秀", IF(B1>80, "良好", "一般")) | 嵌套判断 |

#### 成绩等级判断

```
=IF(A1>=90, "优秀", IF(A1>=80, "良好", IF(A1>=60, "及格", "不及格")))
```

### 3.8 排名函数 RANK

#### 功能

返回某数值在一组数值中的排名。

#### 语法

```
=RANK(数值, 区域, [排序方式])
```

| 参数 | 说明 |
|------|------|
| 数值 | 要排名的数值 |
| 区域 | 参与排名的数据区域 |
| 排序方式 | 0 或省略：降序；1：升序 |

#### 示例

| 公式 | 说明 |
|------|------|
| =RANK(A1, $A$1:$A$10, 0) | A1 在区域中降序排名 |
| =RANK(A1, $A$1:$A$10, 1) | A1 在区域中升序排名 |

### 3.9 函数速查表

| 函数 | 功能 | 示例 |
|------|------|------|
| SUM | 求和 | =SUM(A1:A10) |
| AVERAGE | 平均值 | =AVERAGE(A1:A10) |
| MAX | 最大值 | =MAX(A1:A10) |
| MIN | 最小值 | =MIN(A1:A10) |
| COUNT | 计数（数值） | =COUNT(A1:A10) |
| COUNTA | 计数（非空） | =COUNTA(A1:A10) |
| COUNTIF | 条件计数 | =COUNTIF(A1:A10,">60") |
| IF | 条件判断 | =IF(A1>60,"及格","不及格") |
| RANK | 排名 | =RANK(A1,A1:A10) |

## 📈 第四部分：图表制作

### 4.1 图表概念

**图表**是将数据以图形方式展示，使数据更加直观、易于理解。

### 4.2 创建图表

#### 基本步骤

1. 选择要创建图表的数据区域
2. 点击【插入】→【图表】
3. 选择图表类型
4. 调整图表样式

### 4.3 常见图表类型

#### 柱形图

- 用途：比较不同类别的数据大小
- 特点：垂直的柱形条

```
    ┌───┐
    │   │     ┌───┐
    │   │     │   │
┌───┤   ├───┐ │   │
│   │   │   │ │   │
┴───┴───┴───┴─┴───┴──
  A   B   C   D
```

#### 条形图

- 用途：比较不同类别的数据大小
- 特点：水平的条形

#### 折线图

- 用途：显示数据的变化趋势
- 特点：用线连接各数据点

```
    ●──────●
   /        \
  ●          ●
 /            \
●              ●
└──────────────┘
```

#### 饼图

- 用途：显示各部分占总体的比例
- 特点：圆形，分成若干扇形

```
    ┌───┐
   /│ A │\
  / │   │ \
 │  ├───┤  │
  \ │ B │ /
   \│   │/
    └───┘
```

#### 散点图

- 用途：显示两个变量之间的关系
- 特点：用点表示数据

### 4.4 图表组成

| 组成部分 | 说明 |
|----------|------|
| 图表标题 | 图表的名称 |
| 图表区 | 整个图表区域 |
| 绘图区 | 显示数据的区域 |
| 数据系列 | 图表中的数据 |
| 坐标轴 | X轴（分类轴）和 Y轴（数值轴） |
| 图例 | 说明各数据系列 |
| 数据标签 | 显示具体数值 |
| 网格线 | 辅助阅读的线条 |

### 4.5 编辑图表

#### 添加图表元素

1. 选中图表
2. 点击图表右上角的 + 号
3. 勾选要添加的元素

#### 更改图表类型

1. 选中图表
2. 右键 → 更改图表类型
3. 选择新的图表类型

#### 设置图表样式

1. 选中图表
2. 点击【图表设计】→【图表样式】
3. 选择预设样式

### 4.6 图表选择指南

| 数据特点 | 推荐图表 |
|----------|----------|
| 比较数据大小 | 柱形图、条形图 |
| 显示变化趋势 | 折线图 |
| 显示占比关系 | 饼图 |
| 显示变量关系 | 散点图 |
