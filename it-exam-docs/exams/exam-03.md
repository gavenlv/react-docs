## 四、简答题（每题5分，共20分）

**1. 操作系统的功能：**

1. **处理器管理**：分配和管理 CPU 资源（1分）
2. **存储管理**：管理内存资源（1分）
3. **设备管理**：管理外部设备（1分）
4. **文件管理**：管理文件和文件夹（1分）
5. **用户界面**：提供人机交互界面（1分）

**2. 网络钓鱼：**

网络钓鱼是指攻击者伪装成可信的网站或机构，诱骗用户输入敏感信息（如账号、密码、银行卡号等）。（2分）

识别方法：
1. 检查网址：正规网站网址正确，钓鱼网站网址相似但有差异（1分）
2. 查看证书：点击地址栏锁图标查看 SSL 证书（0.5分）
3. 注意内容：钓鱼网站可能有错别字、排版混乱（0.5分）
4. 警惕要求：正规网站不会要求输入敏感信息（1分）

**3. 文明上网的做法：**

1. 不在网络上散布谣言，转发信息前先核实（1分）
2. 不进行网络暴力，不恶意评论、人身攻击（1分）
3. 不传播不良信息，包括暴力、色情、赌博内容（1分）
4. 尊重他人隐私，不公开他人隐私信息（1分）
5. 尊重知识产权，不传播盗版软件、影视、音乐（1分）

**4. for 循环：**

基本语法：
```python
for 变量 in 序列:
    代码块
```
（2分）

执行过程：
1. 从序列中依次取出每个元素（1分）
2. 将元素赋值给循环变量（1分）
3. 执行代码块（0.5分）
4. 序列遍历完毕后结束循环（0.5分）

## 六、编程题（每题5分，共10分）

**1. 判断奇偶数：**

```python
num = int(input("请输入一个整数："))   # 1.5分
if num % 2 == 0:                      # 1.5分
    print(f"{num}是偶数")              # 1分
else:
    print(f"{num}是奇数")              # 1分
```

**2. 返回列表最大值的函数：**

```python
def find_max(numbers):                # 1分
    max_val = numbers[0]              # 1分
    for num in numbers:               # 1分
        if num > max_val:
            max_val = num
    return max_val                     # 1分

nums = [3, 7, 2, 9, 1, 5]
print(find_max(nums))                  # 1分
```

或使用内置函数：

```python
def find_max(numbers):
    return max(numbers)               # 5分
```
