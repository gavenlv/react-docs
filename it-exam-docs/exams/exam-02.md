## 四、简答题（每题5分，共20分）

**1. 信息获取的基本过程：**

1. **定位信息需求**：明确需要什么信息（1分）
2. **选择信息来源**：确定从哪里获取信息（1分）
3. **采集信息**：通过适当方法获取信息（1分）
4. **保存信息**：将获取的信息保存下来（1分）
5. **评价信息**：判断信息的价值（1分）

**2. 计算机病毒：**

计算机病毒是一种能够自我复制、传播，对计算机系统造成破坏的程序代码。（2分）

特点：
1. 传染性：能自我复制传播（1分）
2. 破坏性：破坏数据或系统（0.5分）
3. 隐蔽性：不易被发现（0.5分）
4. 潜伏性：可能长期隐藏（0.5分）
5. 可触发性：满足条件时激活（0.5分）

**3. 云存储：**

云存储是通过互联网将数据存储在远程服务器上的服务。（2分）

主要优势：
1. 随时随地访问：任何设备、地点都可访问（1分）
2. 自动同步：多设备间数据自动同步（0.5分）
3. 数据安全：专业服务器备份（0.5分）
4. 节省空间：不占用本地存储（0.5分）
5. 方便分享：可生成分享链接（0.5分）

**4. break 和 continue 的区别：**

| 语句 | 作用 | 执行后 |
|------|------|--------|
| break | 跳出整个循环（2分） | 不再执行循环（0.5分） |
| continue | 跳过本次循环（2分） | 继续下一次循环（0.5分） |

## 五、操作题（每题5分，共10分）

**1. PowerPoint 设置切换效果：**

1. 选中要设置切换效果的幻灯片（1分）
2. 点击【切换】选项卡（1分）
3. 在"切换到此幻灯片"组中选择切换效果（1分）
4. 可以设置效果选项、持续时间（1分）
5. 点击"全部应用"可应用到所有幻灯片（1分）

**2. Word 插入页码：**

1. 点击【插入】选项卡（1分）
2. 点击"页码"按钮（1分）
3. 选择页码位置（页面顶端/底端/页边距）（1分）
4. 选择页码样式（1分）
5. 点击"关闭页眉和页脚"或双击正文区域退出编辑（1分）
