## 📊 第一部分：表格操作

### 1.1 插入表格

#### 方法一：快速插入

1. 点击【插入】选项卡
2. 点击【表格】按钮
3. 在网格中移动鼠标选择行列数
4. 点击确认

#### 方法二：对话框插入

1. 点击【插入】→【表格】→【插入表格】
2. 在对话框中设置列数和行数
3. 点击确定

#### 方法三：绘制表格

1. 点击【插入】→【表格】→【绘制表格】
2. 鼠标变成笔形状
3. 拖动绘制表格边框
4. 绘制内部线条

### 1.2 表格基本概念

| 概念 | 说明 |
|------|------|
| 单元格 | 表格中的每一个小格子 |
| 行 | 水平方向的一排单元格 |
| 列 | 垂直方向的一排单元格 |
| 表格区域 | 选中的单元格范围 |

### 1.3 选择表格元素

| 选择对象 | 操作方法 |
|----------|----------|
| 单个单元格 | 点击单元格左侧边框 |
| 整行 | 点击行左侧空白处 |
| 整列 | 点击列顶部边框 |
| 整个表格 | 点击表格左上角十字箭头 |
| 连续单元格 | 拖动鼠标或 Shift+点击 |
| 不连续单元格 | Ctrl+点击 |

### 1.4 编辑表格

#### 插入行/列

| 操作 | 方法 |
|------|------|
| 在上方插入行 | 右键 → 插入 → 在上方插入行 |
| 在下方插入行 | 右键 → 插入 → 在下方插入行 |
| 在左侧插入列 | 右键 → 插入 → 在左侧插入列 |
| 在右侧插入列 | 右键 → 插入 → 在右侧插入列 |

#### 删除行/列

1. 选中要删除的行或列
2. 右键 → 删除单元格
3. 选择删除方式

#### 合并单元格

1. 选中要合并的多个单元格
2. 右键 → 合并单元格
3. 或使用【布局】→【合并单元格】

#### 拆分单元格

1. 选中要拆分的单元格
2. 右键 → 拆分单元格
3. 设置拆分的列数和行数

### 1.5 表格格式设置

#### 边框设置

1. 选中表格
2. 右键 → 边框和底纹
3. 选择边框样式、颜色、宽度
4. 预览并确定

#### 底纹设置

1. 选中单元格
2. 右键 → 边框和底纹 → 底纹
3. 选择填充颜色
4. 确定应用

#### 单元格对齐

| 对齐方式 | 说明 |
|----------|------|
| 靠上两端对齐 | 文字在左上角 |
| 靠上居中对齐 | 文字在顶部居中 |
| 靠上右对齐 | 文字在右上角 |
| 中部两端对齐 | 文字在左侧中部 |
| 中部居中对齐 | 文字在正中间 |
| 中部右对齐 | 文字在右侧中部 |
| 靠下两端对齐 | 文字在左下角 |
| 靠下居中对齐 | 文字在底部居中 |
| 靠下右对齐 | 文字在右下角 |

#### 调整行高和列宽

| 方法 | 操作 |
|------|------|
| 手动调整 | 拖动行或列的边界线 |
| 精确设置 | 布局 → 单元格大小 → 输入数值 |
| 自动调整 | 布局 → 自动调整 → 根据内容自动调整 |

### 1.6 表格实例

创建一个课程表：

| 时间 | 星期一 | 星期二 | 星期三 | 星期四 | 星期五 |
|------|--------|--------|--------|--------|--------|
| 第1节 | 语文 | 数学 | 英语 | 物理 | 化学 |
| 第2节 | 数学 | 语文 | 物理 | 英语 | 语文 |
| 第3节 | 英语 | 物理 | 数学 | 语文 | 数学 |
| 第4节 | 物理 | 英语 | 语文 | 数学 | 英语 |

## 🖼️ 第二部分：图文混排

### 2.1 插入图片

#### 从文件插入

1. 点击【插入】→【图片】→【此设备】
2. 选择图片文件
3. 点击插入

#### 从在线插入

1. 点击【插入】→【图片】→【联机图片】
2. 搜索或选择图片
3. 点击插入

### 2.2 图片环绕方式

| 环绕方式 | 效果说明 | 适用场景 |
|----------|----------|----------|
| 嵌入型 | 图片作为字符处理，随文字移动 | 简单文档 |
| 四周型 | 文字环绕图片四周 | 图文混排 |
| 紧密型 | 文字紧贴图片轮廓 | 不规则图片 |
| 穿越型 | 文字穿过图片空白区域 | 特殊排版 |
| 上下型 | 文字只在图片上下方 | 图文分离 |
| 衬于文字下方 | 图片作为背景 | 水印效果 |
| 浮于文字上方 | 图片遮挡文字 | 特殊效果 |

#### 设置环绕方式

1. 选中图片
2. 点击图片旁边的布局选项按钮
3. 选择环绕方式
4. 或右键 → 环绕文字 → 选择方式

### 2.3 图片编辑

#### 调整大小

| 方法 | 操作 |
|------|------|
| 手动调整 | 拖动图片角点或边中点 |
| 精确设置 | 格式 → 高度/宽度 → 输入数值 |
| 保持比例 | 按住 Shift 拖动角点 |

#### 裁剪图片

1. 选中图片
2. 点击【格式】→【裁剪】
3. 拖动裁剪框调整
4. 点击空白处确认

#### 图片样式

1. 选中图片
2. 点击【格式】→【图片样式】
3. 选择预设样式
4. 或自定义边框、效果

### 2.4 插入形状

#### 插入方法

1. 点击【插入】→【形状】
2. 选择形状类型
3. 在文档中拖动绘制

#### 形状类型

| 类型 | 包含形状 |
|------|----------|
| 线条 | 直线、箭头、曲线 |
| 基本形状 | 矩形、圆形、三角形 |
| 箭头 | 各种箭头形状 |
| 流程图 | 流程图符号 |
| 标注 | 气泡、标注框 |

#### 形状编辑

| 操作 | 方法 |
|------|------|
| 添加文字 | 右键形状 → 添加文字 |
| 设置填充 | 格式 → 形状填充 |
| 设置轮廓 | 格式 → 形状轮廓 |
| 设置效果 | 格式 → 形状效果 |

### 2.5 插入艺术字

#### 插入方法

1. 点击【插入】→【艺术字】
2. 选择艺术字样式
3. 输入文字内容

#### 编辑艺术字

| 操作 | 方法 |
|------|------|
| 修改文字 | 点击艺术字直接编辑 |
| 更改样式 | 格式 → 艺术字样式 |
| 设置填充 | 格式 → 文本填充 |
| 设置轮廓 | 格式 → 文本轮廓 |
| 设置效果 | 格式 → 文本效果 |

### 2.6 插入文本框

#### 插入方法

1. 点击【插入】→【文本框】
2. 选择内置样式或绘制文本框
3. 输入文字内容

#### 文本框特点

- 可以自由移动位置
- 可以设置环绕方式
- 可以设置边框和填充
- 可以链接多个文本框

## 📄 第三部分：页面设置

### 3.1 页边距

#### 页边距概念

页边距是页面内容与纸张边缘之间的空白区域。

| 页边距类型 | 说明 | 用途 |
|------------|------|------|
| 普通 | 上下 2.54cm，左右 3.17cm | 默认设置 |
| 窄 | 上下左右均为 1.27cm | 节省纸张 |
| 适中 | 上下 2.54cm，左右 1.91cm | 平衡排版 |
| 宽 | 上下左右均为 2.54cm | 装订预留 |
| 自定义 | 自行设置数值 | 特殊需求 |

#### 设置页边距

1. 点击【布局】→【页边距】
2. 选择预设或自定义边距
3. 在对话框中输入精确数值

### 3.2 纸张方向

| 方向 | 说明 | 适用场景 |
|------|------|----------|
| 纵向 | 高度大于宽度 | 普通文档、报告 |
| 横向 | 宽度大于高度 | 表格、图片、海报 |

#### 设置纸张方向

1. 点击【布局】→【纸张方向】
2. 选择纵向或横向

### 3.3 纸张大小

| 纸张规格 | 尺寸 | 用途 |
|----------|------|------|
| A4 | 210mm × 297mm | 常用文档 |
| A3 | 297mm × 420mm | 海报、表格 |
| B5 | 176mm × 250mm | 书籍、笔记本 |
| Letter | 216mm × 279mm | 美国标准 |
| 16K | 184mm × 260mm | 国内常用 |

#### 设置纸张大小

1. 点击【布局】→【纸张大小】
2. 选择预设纸张或自定义尺寸

### 3.4 分栏设置

#### 分栏类型

| 类型 | 说明 |
|------|------|
| 一栏 | 默认设置 |
| 两栏 | 文字分两列显示 |
| 三栏 | 文字分三列显示 |
| 左/右 | 不对称分栏 |
| 自定义 | 设置栏数、宽度和间距 |

#### 设置分栏

1. 选中要分栏的文字
2. 点击【布局】→【分栏】
3. 选择分栏类型

### 3.5 分页与分节

#### 分页符

强制从下一页开始新内容。

| 插入方法 | 操作 |
|----------|------|
| 快捷键 | Ctrl + Enter |
| 菜单 | 插入 → 分页符 |

#### 分节符

将文档分成不同的节，每节可以有独立的页面设置。

| 分节类型 | 说明 |
|----------|------|
| 下一页 | 新节从下一页开始 |
| 连续 | 新节紧接当前内容 |
| 偶数页 | 新节从下一个偶数页开始 |
| 奇数页 | 新节从下一个奇数页开始 |

## 📑 第四部分：页眉页脚

### 4.1 页眉页脚概念

| 概念 | 位置 | 用途 |
|------|------|------|
| 页眉 | 页面顶部 | 文档标题、章节名 |
| 页脚 | 页面底部 | 页码、日期、作者 |

### 4.2 插入页眉页脚

#### 插入方法

1. 点击【插入】→【页眉】或【页脚】
2. 选择内置样式或编辑页眉
3. 输入内容
4. 点击【关闭页眉和页脚】

#### 编辑内容

| 内容 | 插入方法 |
|------|----------|
| 文字 | 直接输入 |
| 页码 | 页眉页脚工具 → 页码 |
| 日期 | 页眉页脚工具 → 日期 |
| 图片 | 页眉页脚工具 → 图片 |

### 4.3 页码设置

#### 插入页码

1. 点击【插入】→【页码】
2. 选择位置：页面顶端、页面底端、页边距
3. 选择样式

#### 页码格式

| 设置 | 说明 |
|------|------|
| 编号格式 | 1, 2, 3... 或 一, 二, 三... |
| 起始页码 | 设置起始编号 |
| 包含章节号 | 显示章节编号 |

#### 设置页码格式

1. 点击【插入】→【页码】→【设置页码格式】
2. 选择编号格式
3. 设置起始页码

### 4.4 特殊设置

#### 首页不同

文档首页使用不同的页眉页脚。

1. 双击页眉页脚区域
2. 勾选【设计】→【首页不同】
3. 单独设置首页内容

#### 奇偶页不同

奇数页和偶数页使用不同的页眉页脚。

1. 双击页眉页脚区域
2. 勾选【设计】→【奇偶页不同】
3. 分别设置奇数页和偶数页内容

#### 删除页眉页脚

1. 点击【插入】→【页眉】→【删除页眉】
2. 或【插入】→【页脚】→【删除页脚】

## 📋 答案

### 一、选择题

| 题号 | 答案 | 解析 |
|------|------|------|
| 1 | C | Word 可以通过菜单插入表格，也可以手动绘制表格 |
| 2 | B | 合并单元格前必须先选中要合并的所有单元格 |
| 3 | C | 衬于文字下方使图片作为背景显示 |
| 4 | B | 艺术字在插入选项卡中 |
| 5 | C | 页边距设置在布局选项卡中 |
| 6 | A | A4 纸张尺寸为 210mm × 297mm |
| 7 | A | 页码在插入选项卡中插入 |
| 8 | D | 页眉页脚可以设置首页不同或奇偶页不同 |
