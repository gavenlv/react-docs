## 📖 第一部分：计算机系统概述

### 1.1 计算机系统的组成

计算机系统由**硬件系统**和**软件系统**两大部分组成：

```
┌─────────────────────────────────────────────────────────┐
│                    计算机系统                            │
├─────────────────────────┬───────────────────────────────┤
│       硬件系统           │          软件系统             │
├─────────────────────────┼───────────────────────────────┤
│  • 输入设备              │  • 系统软件                   │
│  • 输出设备              │    - 操作系统                 │
│  • 存储器                │    - 语言处理程序             │
│  • 运算器                │  • 应用软件                   │
│  • 控制器                │    - 办公软件                 │
│                          │    - 图像处理软件             │
└─────────────────────────┴───────────────────────────────┘
```

### 1.2 计算机的特点

| 特点 | 说明 | 举例 |
|------|------|------|
| **运算速度快** | 每秒可进行数亿次运算 | 天气预报、科学计算 |
| **计算精度高** | 可精确到小数点后多位 | 航天计算、金融计算 |
| **存储容量大** | 可存储海量数据 | 数据库、云存储 |
| **具有逻辑判断能力** | 能进行逻辑运算和判断 | 人工智能、自动控制 |
| **自动化程度高** | 无需人工干预可自动执行 | 自动化生产线 |

### 1.3 计算机的应用领域

| 领域 | 应用 | 说明 |
|------|------|------|
| **科学计算** | 气象预报、航天计算 | 复杂的数学运算 |
| **数据处理** | 数据库管理、统计分析 | 处理大量数据 |
| **过程控制** | 工业自动化、智能家居 | 实时监控和控制 |
| **计算机辅助** | CAD、CAM、CAI | 设计、制造、教学 |
| **人工智能** | 语音识别、图像识别 | 模拟人类智能 |
| **网络通信** | 电子邮件、视频会议 | 信息传递和交流 |

## 💻 第二部分：计算机硬件系统

### 2.1 冯·诺依曼体系结构

**冯·诺依曼**（John von Neumann）提出了现代计算机的基本结构，称为"存储程序式计算机"。

#### 冯·诺依曼体系结构的三大核心思想

1. **采用二进制**：计算机内部使用二进制（0和1）表示数据
2. **存储程序**：程序和数据一起存储在内存中
3. **五大部件**：计算机由五大功能部件组成

#### 五大组成部分

```
                    ┌──────────┐
                    │  输入设备  │
                    └─────┬────┘
                          │
                          ▼
┌──────────┐        ┌──────────┐        ┌──────────┐
│  存储器   │◄──────►│  控制器   │◄──────►│  运算器   │
└──────────┘        └─────┬────┘        └──────────┘
                          │
                          ▼
                    ┌──────────┐
                    │  输出设备  │
                    └──────────┘
```

### 2.2 输入设备

**定义**：将外部信息输入到计算机中的设备。

| 设备 | 功能 | 特点 |
|------|------|------|
| **键盘** | 输入文字、命令 | 最基本的输入设备 |
| **鼠标** | 定位、选择操作 | 图形界面必备 |
| **扫描仪** | 将纸质文档数字化 | 输入图片、文字 |
| **摄像头** | 捕捉视频图像 | 视频通话、监控 |
| **麦克风** | 输入声音 | 语音输入、录音 |
| **触摸屏** | 触摸输入 | 兼具输入和输出功能 |

### 2.3 输出设备

**定义**：将计算机处理结果输出的设备。

| 设备 | 功能 | 特点 |
|------|------|------|
| **显示器** | 显示文字、图像 | 最基本的输出设备 |
| **打印机** | 打印文档、图片 | 输出到纸张 |
| **音箱/耳机** | 输出声音 | 音频输出 |
| **投影仪** | 投影显示 | 教学、会议 |

**注意**：触摸屏既是输入设备，也是输出设备。

### 2.4 存储器

**定义**：用于存储程序和数据的部件。

#### 存储器分类

```
存储器
├── 内存（主存储器）
│   ├── ROM（只读存储器）
│   └── RAM（随机存取存储器）
└── 外存（辅助存储器）
    ├── 硬盘（HDD/SSD）
    ├── U盘
    ├── 光盘
    └── SD卡
```

#### 内存与外存的比较

| 特性 | 内存（RAM） | 外存（硬盘） |
|------|-------------|--------------|
| **速度** | 快 | 慢 |
| **容量** | 小（几GB到几十GB） | 大（几百GB到几TB） |
| **价格** | 高 | 低 |
| **断电后** | 数据丢失 | 数据不丢失 |
| **用途** | 暂存正在运行的程序和数据 | 永久存储数据 |

#### ROM 与 RAM 的比较

| 特性 | ROM（只读存储器） | RAM（随机存取存储器） |
|------|-------------------|----------------------|
| **读写性** | 只能读，不能写 | 可读可写 |
| **断电后** | 数据不丢失 | 数据丢失 |
| **用途** | 存储BIOS等固定程序 | 运行程序时暂存数据 |
| **容量** | 较小 | 较大 |

#### 常见外存设备

| 设备 | 容量 | 特点 |
|------|------|------|
| **机械硬盘（HDD）** | 500GB-4TB | 容量大、价格低、速度慢 |
| **固态硬盘（SSD）** | 128GB-2TB | 速度快、价格高 |
| **U盘** | 8GB-256GB | 便携、即插即用 |
| **光盘** | 700MB-50GB | 存储稳定、便于分发 |
| **SD卡** | 8GB-512GB | 用于相机、手机等 |

### 2.5 运算器

**定义**：执行算术运算和逻辑运算的部件。

#### 主要功能

| 运算类型 | 说明 | 举例 |
|----------|------|------|
| **算术运算** | 加、减、乘、除 | 3 + 5 = 8 |
| **逻辑运算** | 与、或、非、异或 | True AND False = False |

#### 组成部分

- **算术逻辑单元（ALU）**：执行运算
- **寄存器**：暂存操作数和结果

### 2.6 控制器

**定义**：指挥和协调计算机各部件工作的部件。

#### 主要功能

1. **取指令**：从内存中取出指令
2. **分析指令**：分析指令的含义
3. **执行指令**：发出控制信号执行指令

#### 组成部分

- **指令寄存器**：存放当前指令
- **程序计数器**：存放下一条指令地址
- **指令译码器**：分析指令含义

### 2.7 中央处理器（CPU）

**CPU**（Central Processing Unit）是计算机的核心部件，由**运算器**和**控制器**组成。

#### CPU 的主要性能指标

| 指标 | 说明 | 影响 |
|------|------|------|
| **主频** | CPU的时钟频率，单位GHz | 主频越高，速度越快 |
| **核心数** | CPU的核心数量 | 核心越多，多任务能力越强 |
| **缓存** | CPU内部的高速存储 | 缓存越大，性能越好 |
| **字长** | CPU一次处理的二进制位数 | 字长越大，处理能力越强 |

#### 常见CPU品牌

- **Intel**：Core i3/i5/i7/i9 系列
- **AMD**：Ryzen 系列

### 2.8 总线

**总线**（Bus）是计算机各部件之间传输信息的公共通道。

#### 总线分类

| 类型 | 功能 | 传输内容 |
|------|------|----------|
| **数据总线** | 传输数据 | 数据信息 |
| **地址总线** | 传输地址 | 存储单元地址 |
| **控制总线** | 传输控制信号 | 控制命令 |

## 🔄 第四部分：计算机工作原理

### 4.1 存储程序原理

**存储程序原理**是冯·诺依曼计算机的核心思想：

1. 程序和数据以二进制形式存储在内存中
2. 计算机自动地从内存中取出指令执行
3. 无需人工干预，计算机可以自动运行程序

### 4.2 计算机执行程序的过程

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   ┌─────────┐    ┌─────────┐    ┌─────────┐            │
│   │ 取指令   │ →  │ 分析指令 │ →  │ 执行指令 │            │
│   └─────────┘    └─────────┘    └─────────┘            │
│        ↑                               │               │
│        └───────────────────────────────┘               │
│                    (重复执行)                           │
└─────────────────────────────────────────────────────────┘
```

#### 执行步骤详解

1. **取指令**：控制器从内存中取出下一条指令
2. **分析指令**：控制器分析指令的含义（做什么操作）
3. **执行指令**：控制器发出控制信号，运算器执行操作
4. **重复**：继续取下一条指令，直到程序结束

### 4.3 指令的组成

一条指令通常由两部分组成：

```
┌────────────────┬────────────────┐
│    操作码      │    操作数      │
└────────────────┴────────────────┘
```

| 部分 | 说明 |
|------|------|
| **操作码** | 指明要执行的操作（如加、减、传送） |
| **操作数** | 指明操作的数据或数据地址 |

## 📜 第五部分：计算机发展历程

### 5.1 计算机的发展阶段

| 代次 | 时间 | 主要元件 | 特点 |
|------|------|----------|------|
| **第一代** | 1946-1957 | 电子管 | 体积大、耗电多、速度慢 |
| **第二代** | 1958-1964 | 晶体管 | 体积减小、可靠性提高 |
| **第三代** | 1965-1971 | 中小规模集成电路 | 体积更小、性能更好 |
| **第四代** | 1972至今 | 大规模集成电路 | 微型化、高性能 |

### 5.2 第一台电子计算机

- **名称**：ENIAC（埃尼阿克）
- **时间**：1946年
- **地点**：美国宾夕法尼亚大学
- **特点**：重30吨，占地170平方米，每秒5000次运算

### 5.3 计算机发展趋势

| 趋势 | 说明 |
|------|------|
| **巨型化** | 超级计算机，用于科学计算 |
| **微型化** | 笔记本、平板、智能手机 |
| **网络化** | 互联网、物联网 |
| **智能化** | 人工智能、机器学习 |

### 四、简答题

**1. 内存和外存的主要区别：**

| 特性 | 内存 | 外存 |
|------|------|------|
| 速度 | 快 | 慢 |
| 容量 | 小 | 大 |
| 价格 | 高 | 低 |
| 断电后 | 数据丢失（RAM） | 数据不丢失 |
| 用途 | 暂存运行中的程序和数据 | 永久存储数据 |

**2. 常见输入输出设备：**

输入设备：键盘、鼠标、扫描仪、摄像头、麦克风
输出设备：显示器、打印机、音箱、投影仪

**3. 操作系统的定义和常见类型：**

操作系统是管理计算机硬件和软件资源的系统软件，是用户和计算机之间的接口。

常见操作系统：
- Windows（个人电脑）
- macOS（苹果电脑）
- Linux（服务器）
- Android（智能手机）
- iOS（iPhone）
