## 📊 第一部分：Excel 简介

### 1.1 Excel 概述

**Microsoft Excel** 是微软公司开发的电子表格软件，是目前最流行的数据处理工具之一。

#### Excel 的主要功能

| 功能 | 说明 |
|------|------|
| 数据存储 | 以表格形式存储数据 |
| 数据计算 | 使用公式和函数进行计算 |
| 数据分析 | 排序、筛选、分类汇总 |
| 图表制作 | 创建各种类型的图表 |
| 数据可视化 | 直观展示数据 |

### 1.2 Excel 界面

```
┌─────────────────────────────────────────────────────────┐
│  标题栏：文件名 - Microsoft Excel                        │
├─────────────────────────────────────────────────────────┤
│  功能区：开始 | 插入 | 页面布局 | 公式 | 数据 | 审阅 | 视图│
├─────────────────────────────────────────────────────────┤
│  名称框：A1    │  编辑栏：显示单元格内容                  │
├───┬───┬───┬───┬───┬───┬───┬───┬───┬───┬─────────────────┤
│   │ A │ B │ C │ D │ E │ F │ G │...│   列标（字母）      │
├───┼───┼───┼───┼───┼───┼───┼───┼───┴─────────────────────┤
│ 1 │   │   │   │   │   │   │   │                         │

```

### 1.3 基本概念

| 概念 | 说明 | 示例 |
|------|------|------|
| 工作簿 | Excel 文件，扩展名 .xlsx | 成绩表.xlsx |
| 工作表 | 工作簿中的表格，默认 Sheet1、Sheet2... | Sheet1 |
| 单元格 | 行列交叉的格子 | A1、B2 |
| 单元格地址 | 列标+行号，唯一标识单元格 | A1、C5 |
| 活动单元格 | 当前选中的单元格 | 有黑色边框 |
| 编辑栏 | 显示和编辑单元格内容 | 名称框右侧 |

### 1.4 行列概念

| 概念 | 说明 | 数量 |
|------|------|------|
| 行 | 水平方向，用数字标识 | 1-1048576 |
| 列 | 垂直方向，用字母标识 | A-XFD（共16384列） |
| 列标 | 列的标识字母 | A, B, C...Z, AA, AB... |

## 📝 第二部分：数据输入

### 2.1 数据类型

#### 数值型数据

| 类型 | 说明 | 示例 |
|------|------|------|
| 整数 | 不带小数的数字 | 100, -50, 0 |
| 小数 | 带小数的数字 | 3.14, -2.5 |
| 百分数 | 带百分号的数字 | 50%, 12.5% |
| 货币 | 带货币符号的数字 | ¥100, $50 |
| 日期 | 日期格式 | 2024/1/1 |
| 时间 | 时间格式 | 12:30:00 |

#### 文本型数据

- 汉字、字母、符号
- 以字母开头或包含文字的数据
- 默认左对齐

#### 公式

- 以 `=` 开头
- 包含运算符、函数、单元格引用
- 示例：`=A1+B1`

### 2.2 输入数据

#### 基本输入

1. 点击目标单元格
2. 直接输入内容
3. 按 Enter 确认或点击其他单元格

#### 在编辑栏输入

1. 点击目标单元格
2. 在编辑栏中输入内容
3. 按 Enter 确认

#### 输入技巧

| 操作 | 方法 |
|------|------|
| 换行输入 | Alt + Enter |
| 取消输入 | Esc |
| 确认输入 | Enter 或 Tab |
| 编辑已有内容 | 双击单元格 或 F2 |

### 2.3 特殊数据输入

#### 输入分数

- 先输入 0 和空格，再输入分数
- 示例：输入 `0 1/2` 显示为 `1/2`

#### 输入身份证号

- 身份证号超过 11 位，会显示为科学计数法
- 解决方法：先输入单引号 `'`，再输入号码
- 示例：`'440105200501011234`

#### 输入日期

| 格式 | 示例 |
|------|------|
| 年/月/日 | 2024/1/1 |
| 年-月-日 | 2024-1-1 |
| 月/日 | 1/1（默认当年） |

#### 输入时间

| 格式 | 示例 |
|------|------|
| 时:分 | 12:30 |
| 时:分:秒 | 12:30:45 |
| 当前时间 | Ctrl + Shift + ; |

### 2.4 自动填充

#### 填充柄

填充柄是单元格右下角的黑色小方块。

```
┌─────┐
│  1  │ ← 选中单元格
└─────┘
     ■ ← 填充柄
```

#### 使用填充柄

1. 选中包含数据的单元格
2. 将鼠标移到填充柄上
3. 鼠标变成黑色十字
4. 拖动到目标区域

#### 填充序列

| 输入内容 | 拖动填充结果 |
|----------|--------------|
| 1 | 1, 1, 1, 1...（复制） |
| 1, 2（选中两个） | 1, 2, 3, 4...（等差序列） |
| 星期一 | 星期二、星期三... |
| 一月 | 二月、三月... |
| 甲 | 乙、丙、丁... |

#### 自定义序列填充

1. 点击【文件】→【选项】→【高级】
2. 找到"编辑自定义列表"
3. 添加自定义序列

### 2.5 快捷输入

| 快捷键 | 功能 |
|--------|------|
| Ctrl + ; | 输入当前日期 |
| Ctrl + Shift + ; | 输入当前时间 |
| Ctrl + D | 向下填充 |
| Ctrl + R | 向右填充 |
| Ctrl + Enter | 在选中区域输入相同内容 |

## 🎨 第三部分：单元格格式

### 3.1 设置单元格格式

#### 打开格式对话框

1. 选中单元格
2. 右键 → 设置单元格格式
3. 或按 Ctrl + 1

### 3.2 数字格式

| 格式类型 | 说明 | 示例 |
|----------|------|------|
| 常规 | 默认格式，无特殊格式 | 123.45 |
| 数值 | 可设置小数位数、千分位 | 1,234.56 |
| 货币 | 添加货币符号 | ¥1,234.56 |
| 会计专用 | 货币符号对齐 | ¥ 1,234.56 |
| 日期 | 日期格式 | 2024年1月1日 |
| 时间 | 时间格式 | 12:30:00 |
| 百分比 | 转换为百分比 | 12.34% |
| 分数 | 分数格式 | 1/2 |
| 科学记数 | 科学计数法 | 1.23E+03 |
| 文本 | 作为文本处理 | 001234 |

#### 设置小数位数

1. 选中单元格
2. 右键 → 设置单元格格式 → 数值
3. 设置小数位数

### 3.3 对齐方式

#### 水平对齐

| 对齐方式 | 说明 |
|----------|------|
| 常规 | 文本左对齐，数值右对齐 |
| 左对齐 | 内容靠左 |
| 居中 | 内容居中 |
| 右对齐 | 内容靠右 |
| 填充 | 重复内容填满单元格 |
| 两端对齐 | 内容分散对齐 |
| 跨列居中 | 内容跨多列居中 |

#### 垂直对齐

| 对齐方式 | 说明 |
|----------|------|
| 靠上 | 内容靠上 |
| 居中 | 内容垂直居中 |
| 靠下 | 内容靠下 |
| 两端对齐 | 内容分散对齐 |

#### 文字控制

| 设置 | 说明 |
|------|------|
| 自动换行 | 文字自动换行显示 |
| 缩小字体填充 | 字体自动缩小适应列宽 |
| 合并单元格 | 将多个单元格合并为一个 |

### 3.4 字体格式

| 设置 | 说明 |
|------|------|
| 字体 | 设置字体类型 |
| 字号 | 设置文字大小 |
| 颜色 | 设置文字颜色 |
| 加粗 | Ctrl + B |
| 倾斜 | Ctrl + I |
| 下划线 | Ctrl + U |
| 删除线 | 添加删除线效果 |

### 3.5 边框设置

#### 添加边框

1. 选中单元格
2. 右键 → 设置单元格格式 → 边框
3. 选择线条样式和颜色
4. 点击预览区域的边框位置

#### 边框样式

| 样式 | 说明 |
|------|------|
| 无边框 | 清除边框 |
| 细边框 | 细实线 |
| 粗边框 | 粗实线 |
| 双线边框 | 双线效果 |

#### 快速设置边框

- 开始 → 边框 → 选择预设边框

### 3.6 填充颜色

#### 设置背景色

1. 选中单元格
2. 右键 → 设置单元格格式 → 填充
3. 选择背景色

#### 设置图案

1. 选择填充对话框中的图案样式
2. 选择图案颜色

## ✏️ 第四部分：编辑操作

### 4.1 选择操作

#### 选择单元格

| 选择对象 | 操作方法 |

#### 快速选择

| 快捷键 | 功能 |
|--------|------|
| Ctrl+A | 选择当前区域 |
| Ctrl+A+A | 选择整个工作表 |

| Ctrl+Shift+Home | 选择到工作表开头 |

### 4.2 编辑内容

#### 修改内容

| 方法 | 操作 |
|------|------|
| 双击单元格 | 进入编辑模式 |

| 点击编辑栏 | 在编辑栏修改 |

#### 清除内容

| 操作 | 方法 |
|------|------|
| 清除内容 | Delete 或 右键→清除内容 |
| 清除格式 | 开始 → 清除 → 清除格式 |
| 清除全部 | 开始 → 清除 → 全部清除 |

### 4.3 复制与移动

#### 复制操作

| 方法 | 操作 |
|------|------|
| 快捷键 | Ctrl + C 复制，Ctrl + V 粘贴 |
| 右键菜单 | 右键 → 复制 |
| 拖动复制 | Ctrl + 拖动单元格边框 |

#### 移动操作

| 方法 | 操作 |
|------|------|
| 快捷键 | Ctrl + X 剪切，Ctrl + V 粘贴 |
| 右键菜单 | 右键 → 剪切 |
| 拖动移动 | 直接拖动单元格边框 |

#### 选择性粘贴

1. 复制单元格
2. 右键 → 选择性粘贴
3. 选择粘贴内容：

| 选项 | 说明 |
|------|------|
| 全部 | 粘贴内容和格式 |
| 公式 | 只粘贴公式 |
| 数值 | 只粘贴数值 |
| 格式 | 只粘贴格式 |
| 转置 | 行列互换 |

### 4.4 插入与删除

#### 插入单元格

1. 右键 → 插入
2. 选择插入方式：
   - 活动单元格右移
   - 活动单元格下移
   - 整行
   - 整列

#### 插入行/列

| 操作 | 方法 |
|------|------|
| 插入行 | 右键行号 → 插入 |
| 插入列 | 右键列标 → 插入 |
| 插入多行 | 选中多行 → 右键 → 插入 |

#### 删除单元格

1. 选中单元格
2. 右键 → 删除
3. 选择删除方式

#### 删除行/列

| 操作 | 方法 |
|------|------|
| 删除行 | 右键行号 → 删除 |
| 删除列 | 右键列标 → 删除 |

### 4.5 行列调整

#### 调整行高

| 方法 | 操作 |
|------|------|
| 手动调整 | 拖动行号下边框 |
| 精确设置 | 右键行号 → 行高 → 输入数值 |
| 自动调整 | 双击行号下边框 |

#### 调整列宽

| 方法 | 操作 |
|------|------|
| 手动调整 | 拖动列标右边框 |
| 精确设置 | 右键列标 → 列宽 → 输入数值 |
| 自动调整 | 双击列标右边框 |

#### 隐藏与显示

| 操作 | 方法 |
|------|------|
| 隐藏行 | 选中行 → 右键 → 隐藏 |
| 隐藏列 | 选中列 → 右键 → 隐藏 |
| 显示行 | 选中隐藏行两侧 → 右键 → 取消隐藏 |
| 显示列 | 选中隐藏列两侧 → 右键 → 取消隐藏 |

### 四、操作题

**操作题1 参考步骤：**

1. 打开 Excel，新建工作簿
2. 点击 A1，输入"学生成绩表"，按 Enter
3. 点击 A2，输入"姓名"，按 Tab 键移动到 B2
4. 依次输入"语文"、"数学"、"英语"、"总分"
5. 在 A3:A5 输入学生姓名
6. 在 B3:D5 输入各科成绩

**操作题2 参考步骤：**

1. 选中 A1:E1，点击【开始】→【合并后居中】
2. 设置字体为黑体、16号、加粗
3. 选中 A2:E2，设置填充颜色为浅蓝色
4. 选中 A1:E5，点击边框按钮，选择所有边框
5. 选中所有单元格，设置居中对齐

**操作题3 参考步骤：**

1. 右键点击第 2 行行号，选择"插入"
2. 右键点击 A 列列标，选择"列宽"，输入 15
3. 右键点击 D 列列标，选择"隐藏"
4. 双击 Sheet1 标签，输入"成绩表"
