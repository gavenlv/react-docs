## 📖 第一部分：操作系统概述

### 1.1 操作系统的定义

**操作系统**（Operating System，简称OS）是管理计算机硬件和软件资源的系统软件，是用户与计算机之间的接口。

### 1.2 操作系统的功能

| 功能 | 说明 | 具体内容 |
|------|------|----------|
| **处理器管理** | 分配和管理CPU资源 | 进程调度、任务管理 |
| **存储管理** | 管理内存空间 | 内存分配、回收 |
| **设备管理** | 管理输入输出设备 | 驱动程序、设备分配 |
| **文件管理** | 管理文件和文件夹 | 创建、复制、删除文件 |
| **用户接口** | 提供用户操作界面 | 图形界面、命令行 |

### 1.3 常见操作系统

| 操作系统 | 开发商 | 特点 | 应用场景 |
|----------|--------|------|----------|
| **Windows** | Microsoft | 图形界面、易用性强 | 个人电脑、办公 |
| **macOS** | Apple | 界面美观、稳定性好 | 苹果电脑 |
| **Linux** | 开源社区 | 开源、安全、稳定 | 服务器、开发 |
| **Android** | Google | 开源、应用丰富 | 智能手机、平板 |
| **iOS** | Apple | 安全、流畅、封闭 | iPhone、iPad |
| **HarmonyOS** | Huawei | 分布式、跨设备 | 华为设备 |

### 1.4 Windows 版本发展

| 版本 | 发布时间 | 特点 |
|------|----------|------|
| Windows XP | 2001年 | 经典版本，广泛使用 |
| Windows 7 | 2009年 | 稳定性好，用户喜爱 |
| Windows 10 | 2015年 | 现代化界面，持续更新 |
| Windows 11 | 2021年 | 最新版本，全新界面 |

## 📁 第三部分：文件与文件夹管理

### 3.1 文件的基本概念

#### 文件的定义

**文件**是存储在存储介质上的相关信息的集合，是数据存储的基本单位。

#### 文件名

文件名由**主文件名**和**扩展名**组成，中间用点号分隔。

```
文件名格式：主文件名.扩展名

示例：报告.docx
      │    │
      │    └── 扩展名（表示文件类型）
      └─────── 主文件名（用户自定义）
```

#### 文件命名规则

| 规则 | 说明 |
|------|------|
| 允许字符 | 字母、数字、汉字、下划线、空格等 |
| 禁止字符 | \ / : * ? " < > | |
| 长度限制 | 最多255个字符 |
| 大小写 | 不区分大小写 |
| 唯一性 | 同一文件夹内不能有同名文件 |

#### 常见文件类型

| 扩展名 | 文件类型 | 打开程序 |
|--------|----------|----------|
| .txt | 文本文件 | 记事本 |
| .docx | Word文档 | Word |
| .xlsx | Excel表格 | Excel |
| .pptx | PowerPoint演示文稿 | PowerPoint |
| .jpg/.png | 图片文件 | 图片查看器 |
| .mp3 | 音频文件 | 音乐播放器 |
| .mp4 | 视频文件 | 视频播放器 |
| .pdf | PDF文档 | PDF阅读器 |
| .zip/.rar | 压缩文件 | 解压软件 |

### 3.2 文件夹的基本概念

#### 文件夹的定义

**文件夹**是用来组织和管理文件的容器，可以包含文件和其他文件夹（子文件夹）。

#### 文件夹结构

```
C:\                          （根目录）
├── Windows\                 （系统文件夹）
├── Program Files\           （程序文件夹）
├── Users\                   （用户文件夹）
│   ├── 用户名\
│   │   ├── Desktop\         （桌面）
│   │   ├── Documents\       （文档）
│   │   ├── Downloads\       （下载）
│   │   ├── Pictures\        （图片）
│   │   └── Music\           （音乐）
```

#### 路径的概念

**路径**是文件或文件夹在计算机中的位置描述。

| 路径类型 | 格式 | 示例 |
|----------|------|------|
| 绝对路径 | 从根目录开始的完整路径 | C:\Users\用户名\Documents\报告.docx |
| 相对路径 | 从当前位置开始的路径 | Documents\报告.docx |

### 3.3 文件和文件夹操作

#### 选定操作

| 操作 | 方法 |
|------|------|
| 选定单个 | 单击 |
| 选定连续多个 | 单击第一个 + Shift + 单击最后一个 |
| 选定不连续多个 | Ctrl + 单击 |
| 选定全部 | Ctrl + A |

#### 新建操作

| 操作 | 方法 |
|------|------|
| 新建文件夹 | 右键 → 新建 → 文件夹 |
| 新建文件 | 右键 → 新建 → 选择文件类型 |

#### 复制操作

| 方法 | 操作步骤 |
|------|----------|
| 方法一 | 右键 → 复制 → 到目标位置 → 粘贴 |
| 方法二 | Ctrl + C（复制）→ Ctrl + V（粘贴） |
| 方法三 | 拖动时按住 Ctrl 键 |

#### 移动操作

| 方法 | 操作步骤 |
|------|----------|
| 方法一 | 右键 → 剪切 → 到目标位置 → 粘贴 |
| 方法二 | Ctrl + X（剪切）→ Ctrl + V（粘贴） |
| 方法三 | 直接拖动（同一磁盘） |

#### 删除操作

| 方法 | 操作步骤 |
|------|----------|
| 方法一 | 选中 → 按 Delete 键 |
| 方法二 | 右键 → 删除 |
| 方法三 | 拖到回收站 |
| 彻底删除 | Shift + Delete（不进回收站） |

#### 重命名操作

| 方法 | 操作步骤 |
|------|----------|
| 方法一 | 右键 → 重命名 |
| 方法二 | 选中 → 按 F2 键 |
| 方法三 | 单击文件名（慢速单击两次） |

### 3.4 回收站

#### 回收站的功能

回收站是临时存放被删除文件的地方，可以恢复误删的文件。

#### 回收站操作

| 操作 | 方法 |
|------|------|
| 还原文件 | 右键文件 → 还原 |
| 彻底删除 | 右键文件 → 删除 |
| 清空回收站 | 右键回收站 → 清空回收站 |

#### 注意事项

- 从U盘、网络位置删除的文件不进入回收站
- 使用 Shift + Delete 删除的文件不进入回收站
- 清空回收站后文件无法恢复

## 🔍 第五部分：文件搜索与属性

### 5.1 文件搜索

#### 搜索方法

1. **使用搜索框**：在文件资源管理器右上角搜索框输入关键词
2. **使用通配符**：
   - `*`：代表任意多个字符
   - `?`：代表任意一个字符

#### 搜索示例

| 搜索内容 | 说明 |
|----------|------|
| *.docx | 搜索所有Word文档 |
| report* | 搜索以report开头的文件 |
| *.jpg | 搜索所有JPG图片 |
| a?c.txt | 搜索如abc.txt、a1c.txt等 |

### 5.2 文件属性

#### 查看文件属性

右键文件 → 属性

#### 常见属性信息

| 属性 | 说明 |
|------|------|
| 类型 | 文件类型 |
| 位置 | 文件存储路径 |
| 大小 | 文件占用空间 |
| 创建时间 | 文件创建的日期和时间 |
| 修改时间 | 文件最后修改的日期和时间 |
| 只读 | 文件只能读取，不能修改 |
| 隐藏 | 文件不显示在列表中 |

#### 设置文件属性

1. 右键文件 → 属性
2. 勾选"只读"或"隐藏"
3. 点击"确定"

### 四、简答题

**1. 复制和移动的区别：**

- **复制**：原位置保留文件，在新位置创建文件的副本。文件数量增加。
- **移动**：原位置的文件被删除，在新位置创建文件。文件数量不变，只是位置改变。

**2. 常用Windows快捷键：**

| 快捷键 | 功能 |
|--------|------|
| Ctrl + C | 复制 |
| Ctrl + V | 粘贴 |
| Ctrl + Z | 撤销 |
| Win + D | 显示桌面 |
| Alt + F4 | 关闭窗口 |
| Win + E | 打开此电脑 |
| Ctrl + A | 全选 |

**3. 文件命名规则：**

- 允许使用字母、数字、汉字、下划线、空格等
- 禁止使用的字符：\ / : * ? " < > |
- 文件名长度最多255个字符
- 同一文件夹内不能有同名文件
- 不区分大小写
