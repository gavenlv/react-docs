## 📚 考试概述

### 考试形式

- **考试时间**：60分钟
- **试卷结构**：选择题 + 操作题
- **分值分布**：基础知识 40% + 操作技能 60%

### 考试范围

| 模块 | 内容 | 分值占比 |
|------|------|----------|
| 信息技术基础 | 信息概念、信息处理 | 10% |
| 计算机系统 | 硬件、软件、二进制 | 15% |
| 操作系统 | Windows 基本操作 | 10% |
| 文字处理 | Word 文档编辑 | 15% |
| 电子表格 | Excel 数据处理 | 15% |
| 演示文稿 | PowerPoint 制作 | 10% |
| 计算机网络 | 网络基础、互联网应用 | 10% |
| 信息安全 | 网络安全、法律法规 | 5% |
| 程序设计 | Python 基础编程 | 10% |
| 人工智能（拓展） | AI基础、机器学习、AI伦理 | 选学 |

## 🎯 学习建议

### 学习方法

1. **理解概念**：先理解基本概念，再进行操作练习
2. **多做练习**：操作题需要大量练习，熟能生巧
3. **总结归纳**：整理知识点，建立知识体系
4. **模拟测试**：考前进行模拟测试，熟悉考试形式

### 复习重点

- 二进制与十进制转换
- Word/Excel/PowerPoint 常用操作
- Excel 常用函数（SUM、AVERAGE、MAX、MIN、IF）
- Python 基本语法和简单算法
- 网络安全与法律法规

## 📝 考试技巧

### 选择题技巧

1. 认真审题，注意关键词
2. 排除法缩小范围
3. 不确定的题目先标记，最后再答

### 操作题技巧

1. 先浏览题目要求，规划操作步骤
2. 注意保存文件和文件名
3. 检查操作结果是否符合要求
