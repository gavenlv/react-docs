## 🧠 第一部分：人工智能概述

### 1.1 什么是人工智能

#### 定义

**人工智能（Artificial Intelligence，简称 AI）** 是计算机科学的一个分支，致力于研究和开发能够模拟、延伸和扩展人类智能的理论、方法、技术及应用系统。

#### 理解人工智能

```
┌─────────────────────────────────────────────────────────┐
│                    人工智能（AI）                        │
├─────────────────────────────────────────────────────────┤
│  定义：让机器具备类似人类的智能能力                      │
│                                                         │
│  核心能力：                                              │
│  ├── 学习能力：从数据中获取知识                          │
│  ├── 推理能力：根据已知推断未知                          │
│  ├── 感知能力：识别图像、声音等                          │
│  ├── 理解能力：理解语言、文本                            │
│  └── 决策能力：做出合理选择                              │
└─────────────────────────────────────────────────────────┘
```

#### 人类智能 vs 人工智能

| 特征 | 人类智能 | 人工智能 |
|------|----------|----------|
| 学习方式 | 经验学习、举一反三 | 数据驱动、模式识别 |
| 处理速度 | 相对较慢 | 极快 |
| 记忆能力 | 有限且易遗忘 | 海量存储、精确回忆 |
| 创造能力 | 具有创造性 | 基于已有数据生成 |
| 情感理解 | 具有情感 | 无真实情感 |
| 适用场景 | 复杂、模糊问题 | 明确、结构化任务 |

### 1.2 人工智能的发展历程

#### 发展阶段

```
第一阶段：萌芽期（1950年代）
├── 1950年：图灵提出"图灵测试"
├── 1956年：达特茅斯会议，"人工智能"概念诞生
└── 代表：逻辑推理程序、跳棋程序

第二阶段：成长期（1960-1970年代）
├── 专家系统出现
├── 机器学习概念提出
└── 第一次AI寒冬（算力不足）

第三阶段：发展期（1980-1990年代）
├── 专家系统广泛应用
├── 神经网络研究复兴
└── 第二次AI寒冬（商业化困难）

第四阶段：突破期（2000年代至今）
├── 深度学习技术突破
├── 大数据时代到来
├── 算力大幅提升
└── AlphaGo、ChatGPT等里程碑
```

#### 重要里程碑

| 年份 | 事件 | 意义 |
|------|------|------|
| 1950 | 图灵测试提出 | AI研究的理论起点 |
| 1956 | 达特茅斯会议 | AI学科正式诞生 |
| 1997 | 深蓝战胜国际象棋冠军 | AI在特定领域超越人类 |
| 2011 | Siri 发布 | AI进入日常生活 |
| 2016 | AlphaGo 战胜李世石 | 深度学习的重大突破 |
| 2022 | ChatGPT 发布 | 大语言模型时代开启 |
| 2024 | Sora、GPT-4o 发布 | 多模态AI快速发展 |

#### 按能力分类

│  ├── 定义：专注于单一任务的AI                           │
│  ├── 特点：在特定领域表现出色                           │

#### 按学习方法分类

|------|------|------|

| 强化学习 | 通过奖励机制学习 | 游戏AI、机器人控制 |

## 🔍 第二部分：人工智能核心技术

### 2.1 机器学习

#### 什么是机器学习

**机器学习（Machine Learning）** 是人工智能的核心技术，让计算机能够从数据中自动学习规律，而无需人工编写具体规则。

#### 传统编程 vs 机器学习

```
传统编程：
┌──────────┐     ┌──────────┐     ┌──────────┐
│  输入    │ ──→ │  规则    │ ──→ │  输出    │
│  数据    │     │  （人工）│     │  结果    │
└──────────┘     └──────────┘     └──────────┘

机器学习：
┌──────────┐     ┌──────────┐     ┌──────────┐
│  输入    │ ──→ │  算法    │ ──→ │  规则    │
│  数据    │     │  （自动）│     │  （模型）│
└──────────┘     └──────────┘     └──────────┘
         ↑                              │
         └──────────────────────────────┘
                   反馈学习
```

#### 机器学习的基本流程

```
1. 数据收集
   └── 收集相关数据（如猫狗图片）

2. 数据预处理
   └── 清洗、标准化数据

3. 特征提取
   └── 提取关键特征（如颜色、形状）

4. 模型训练
   └── 使用算法学习规律

5. 模型评估
   └── 测试模型准确率

6. 模型应用
   └── 部署到实际场景
```

### 2.2 深度学习

#### 什么是深度学习

**深度学习（Deep Learning）** 是机器学习的一个分支，模拟人脑神经网络结构，通过多层神经网络自动学习数据的特征表示。

#### 神经网络结构

```
输入层          隐藏层           输出层
  ○              ○
  ○ ────────→    ○  ────────→    ○
  ○              ○               ○
  ○              ○
  ○

数据输入    特征提取        结果输出
（如图像）  （自动学习）    （如分类）
```

#### 深度学习的优势

| 特点 | 说明 |
|------|------|
| 自动特征提取 | 不需要人工设计特征 |
| 处理复杂数据 | 能处理图像、语音等非结构化数据 |
| 高准确率 | 在许多任务上达到或超越人类水平 |
| 可扩展性 | 数据越多，效果越好 |

#### 深度学习的应用

```
计算机视觉
├── 人脸识别：手机解锁、安防监控
├── 图像分类：识别物体种类
├── 目标检测：自动驾驶中的物体识别
└── 图像生成：AI绘画

自然语言处理
├── 机器翻译：Google翻译、百度翻译
├── 智能对话：ChatGPT、文心一言
├── 文本生成：AI写作
└── 情感分析：分析用户评论情感

语音处理
├── 语音识别：语音转文字
├── 语音合成：文字转语音
└── 声纹识别：身份验证
```

### 2.3 自然语言处理（NLP）

#### 什么是自然语言处理

**自然语言处理（Natural Language Processing，NLP）** 是让计算机理解、解释和生成人类语言的技术。

#### NLP的主要任务

```
┌─────────────────────────────────────────────────────────┐
│              自然语言处理主要任务                        │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  理解语言：                                              │
│  ├── 分词：将句子切分成词语                              │
│  │   例："我爱学习" → ["我", "爱", "学习"]              │
│  ├── 词性标注：标注每个词的词性                          │
│  │   例："我/r 爱/v 学习/vn"                           │
│  ├── 命名实体识别：识别人名、地名等                      │
│  │   例："张三在北京工作" → 人名：张三，地名：北京      │
│  └── 情感分析：判断文本的情感倾向                        │
│      例："这部电影很好看" → 正面情感                    │
│                                                         │
│  生成语言：                                              │
│  ├── 机器翻译：将一种语言翻译成另一种                    │
│  ├── 文本摘要：生成文章摘要                              │
│  ├── 问答系统：回答用户问题                              │
│  └── 对话生成：智能对话                                  │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

#### 大语言模型（LLM）

**大语言模型（Large Language Model，LLM）** 是基于深度学习的大规模语言模型，能够理解和生成人类语言。

| 模型 | 开发者 | 特点 |
|------|--------|------|
| GPT系列 | OpenAI | 强大的文本生成能力 |
| 文心一言 | 百度 | 中文理解能力强 |
| 通义千问 | 阿里巴巴 | 多场景应用 |
| 讯飞星火 | 科大讯飞 | 教育领域应用 |
| Claude | Anthropic | 安全性高 |

### 2.4 计算机视觉

#### 什么是计算机视觉

**计算机视觉（Computer Vision）** 是让计算机能够"看懂"图像和视频的技术。

#### 主要任务

```
图像分类
├── 任务：判断图像属于哪个类别
├── 示例：识别图片是猫还是狗
└── 应用：相册自动分类

目标检测
├── 任务：定位图像中的物体并分类
├── 示例：识别图像中的人、车、动物位置
└── 应用：自动驾驶、安防监控

图像分割
├── 任务：精确划分图像中每个像素的类别
├── 示例：医学图像中的器官分割
└── 应用：医学诊断、视频会议背景替换

人脸识别
├── 任务：识别或验证人脸身份
├── 示例：手机人脸解锁
└── 应用：身份验证、安防系统
```

### 3.2 智能交通

#### 自动驾驶

```
自动驾驶等级（SAE标准）：

L0 - 无自动化
└── 完全由人类驾驶

L1 - 驾驶辅助
└── 单一功能辅助（如自适应巡航）

L2 - 部分自动化
└── 多功能辅助，人类仍需监控

L3 - 有条件自动化
└── 特定条件下自动驾驶，需人类接管

L4 - 高度自动化
└── 大多数情况下自动驾驶

L5 - 完全自动化
└── 全场景自动驾驶（尚未实现）
```

### 四、简答题

**1. 人工智能定义和核心能力：**

人工智能是计算机科学的一个分支，致力于研究和开发能够模拟、延伸和扩展人类智能的理论、方法、技术及应用系统。（4分）

核心能力：
- 学习能力：从数据中获取知识（1.5分）
- 推理能力：根据已知推断未知（1分）
- 感知能力：识别图像、声音等（1分）
- 理解能力：理解语言、文本（1分）
- 决策能力：做出合理选择（1.5分）

**2. 机器学习的基本流程：**

1. 数据收集：收集相关数据（2分）
2. 数据预处理：清洗、标准化数据（2分）
3. 特征提取：提取关键特征（2分）
4. 模型训练：使用算法学习规律（2分）
5. 模型评估：测试模型准确率（1分）
6. 模型应用：部署到实际场景（1分）

**3. AI日常应用实例：**

| 应用 | AI技术 |
|------|--------|
| 手机人脸解锁 | 人脸识别/计算机视觉 |
| 语音助手（Siri） | 语音识别、自然语言处理 |
| 抖音视频推荐 | 推荐算法 |
| 扫地机器人 | 路径规划、环境感知 |
| 智能翻译 | 自然语言处理、机器翻译 |
